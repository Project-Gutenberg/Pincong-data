办公体验还凑合，一台电脑带显示器和 WPS 企业版，采购价是五六千，软件生态勉勉强强。

单位之前采购的同价格电脑一般是当代奔腾＋机械硬盘，得益于 Linux 和固态硬盘的原因，国产电脑基础体验还是比非国产好的。

WPS 企业版功能很少，只有基础功能，但是 bug 也比较少，凑合能用。

WPS 个人版功能就多了，还有云盘功能，可惜与企业版不能共存，会相互覆盖。

内网传输用信使，与飞鸽互相兼容，虽然兼容上 bug 一堆，但是凑合能用。

QQ 比较残废，应该是网页版 QQ 套壳，UI 是几十年前那种，非常简陋。

微信有网页套壳版和原生版，网页套壳版 bug 比较少但是停止更新了，原生版在开发，但是更新不勤快，bug 多，而且有很多 bug 严重影响使用。

银河麒麟 V10 和 ubuntu 很像…… 非常像，所以 GIMP 之类的在 ubuntu 上能开箱即用的软件在银河麒麟依旧可以用，就是需要 ARM 包。

系统的卖点之一，KMRE 的体验很差，KMRE 号称可以运行安卓应用，而且安卓应用内的文件可以直接在系统外访问。实际上是真的做到了，只是佳景微 JM7200 的性能太过孱弱，导致运行安卓应用的流畅度很差。

而且运行 KMRE 需要的性能也很高，运行环境和软件会吃掉一半多的 CPU 性能。

目前 KMRE 无法以图形化的方式指定分辨率，比如我想让一个安卓软件以横屏运行，必须通过命令行指定分辨率来启动，而且每次都必须用命令行启动，虽然我是没啥，但是这个电脑面向的应当是普通职工，他们大概率玩不转这个，说实在的，给他们搞个 sh 脚本都担心脚本被玩坏。

浏览器，应用商店有 360 浏览器、火狐浏览器和基于 Chromium 编译的一个浏览器，还有一个奇安信可信浏览器，所以这方面生态是很丰富的。

打印机驱动比较全，我上次去京瓷官网翻老机器的说明书，结果发现那台老机器居然都有银河麒麟的驱动。

不过有些键盘因为固件问题在麒麟上不能使用，比如 cherry 的 mx8.0，不过我去官网折磨客服后，客服丢给我个固件，在 Windows 上给键盘刷了之后就可以正常使用了，给 cherry 点个赞。

其他的，ubuntu 可以搞的银河基本也能搞，比如代理，可以用 clash，就是不能直接开箱即用，有一点折腾。

这台电脑给我感觉是，流畅度还行，很少死机，也很少掉帧，毕竟有固态硬盘保底，比奔腾＋机械硬盘＋Windows 桌面流畅度更高也正常。

专业软件生态比较残，ARM Ubuntu 能用的它都能用，ARM Ubuntu 不能用的它也够呛。

娱乐生态残上加残，至今没找到一个好用的视频播放器，网易云音乐也没有客户端，只能上 github 找了个第三方网易云客户端，叫 YesPlayMusic。

游戏不要想了，不存在的，指望这个打游戏你怕不是在做梦。

**按照上述逻辑推理，我们第一步要盯住全球流动性，从目前来看新冠大流行导致全球货币超发，因此加息初期出现流动性危机的可能性较低**。

第二，欧洲、美国经济必然在第二波加息潮下陷入衰退，比如欧洲央行利率超过 1.5%，美联储利率超过 2.5%，这都将引发经济衰退，经济衰退的标准是汽车、住房消费暴跌，零售额出现短期见顶，就业率见顶开始回落，失业率开始增加。

第三，全球主要国家资本市场必然在加息出现上述，加息末期出现下跌，一旦欧美资本市场反弹结束开始筑顶，那么基本预示着经济见顶。

第四，国内进出口额增速降低，甚至开始走下坡路，开始[负增长](https://www.zhihu.com/search?q=%E8%B4%9F%E5%A2%9E%E9%95%BF&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2597731399%7D)，基本预示着外循环开始紧缩，全球经济进入到衰退期。

Open source
-----------

Linux is also distributed under an open source license. Open source follows these key tenants:

*   The freedom to run the program, for any purpose.
*   The freedom to study how the program works, and change it to make it do what you wish.
*   The freedom to redistribute copies so you can help your neighbor.
*   The freedom to distribute copies of your modified versions to others.

These points are crucial to understanding the community that works together to create the Linux platform. Without a doubt, Linux is an operating system that is “by the people, for the people”. These tenants are also a main factor in why many people choose Linux. It’s about freedom and freedom of use and freedom of choice.

By virtue of its open source licensing, Linux is freely available to anyone. However, the trademark on the name “Linux” rests with its creator, Linus Torvalds. The source code for Linux is under copyright by its many individual authors, and licensed under the [GPLv2 license](https://link.zhihu.com/?target=https%3A//www.gnu.org/licenses/old-licenses/gpl-2.0.en.html).

The term “Linux” technically refers to just the Linux kernel. Most people refer to the entire operating system as "Linux" because to most users an OS includes a bundle of programs, tools, and services (like a desktop, clock, an application menu, and so on). Some people, particularly members of the [Free Software Foundation](https://link.zhihu.com/?target=https%3A//fsf.org/), refer to this collection as GNU/Linux, because many vital tools included are GNU components. However, not all Linux installations use GNU components as a part of the operating system: [Android](https://link.zhihu.com/?target=https%3A//opensource.com/tags/android), for example, uses a Linux kernel but relies very little on GNU tools.
