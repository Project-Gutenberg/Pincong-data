### 知乎用户 萝魏紫​ 发表
    
这是软件安全知识，算在架构里面的一部分。

中国有[数据安全法](https://www.zhihu.com/search?q=%E6%95%B0%E6%8D%AE%E5%AE%89%E5%85%A8%E6%B3%95&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2470739781%7D)，网络安全法，个人信息保护法等几个法令，这其中规定了敏感个人信息的安全收集，储存和销毁。

最主要的定义来自于个人信息保护法 28 条。这个法，其实去年才通过。。。

**第二十八条　敏感个人信息是一旦泄露或者非法使用，容易导致自然人的人格尊严受到侵害或者人身、财产安全受到危害的个人信息，包括生物识别、宗教信仰、特定身份、医疗健康、金融账户、行踪轨迹等信息，以及不满十四周岁未成年人的个人信息。**

IP 信息本身，是属于行踪轨迹的一部分。根据《[关于办理侵犯公民个人信息刑事案件](https://www.zhihu.com/search?q=%E5%85%B3%E4%BA%8E%E5%8A%9E%E7%90%86%E4%BE%B5%E7%8A%AF%E5%85%AC%E6%B0%91%E4%B8%AA%E4%BA%BA%E4%BF%A1%E6%81%AF%E5%88%91%E4%BA%8B%E6%A1%88%E4%BB%B6&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2470739781%7D)适用法律若干问题的解释》IP 地址能够显示特定时间，客体出现在特定场合，这就属于行踪轨迹了。顺便附送一个小知识，暂住地信息，属于静态信息，不属于行踪轨迹。

这部分信息是定义成敏感信息的。而 IP 属地，定位到省，就不一样，由于其没有特定性（防九年制义务教育杠，在法理上，特定是一个特殊词汇，不是字典意义，特定必须能够明确定位到一个具体客体）

原则上来说，虽然软件系统天然能够接触到 IP 信息，但是必须作一次单向不可逆转换，变成属地信息才能展示。而如果有任何软件架构设计有问题，直接记录 IP 在展示时候转化成属地显示，如果被发现，能罚死。
    
    
    
    
### 知乎用户 甄昊元​ 发表
    
因为 IP 地址比属地更为复杂。在同一个城市，可能因为接入的运营商不同，被分配不同的 IP 地址，这就导致一座城市的 IP 地址至少可能出现三个主要的区段。联通、移动、电信各自有不同的 IP 字段头。加上一些杂七杂八的 IP 分配，光靠看 IP 地址，你基本上是无法分辨其地理位置的。

本次 APP 显示 IP 地址，主要目的还是为了清理一些隐瞒身份带节奏、冒充当事人的行为。对这种清理来说，重要的不是显示你具体的网络地址，而是显示你在实体空间中的地理地址。

另一方面，显示所处省份仍然是一种对个人隐私的保护。如果如台湾的 PTT 论坛一样，显示你的 IP 地址，那么就可以获取你精确的地理位置了，比如下面这个 IP 区段，就是河北省衡水市某网吧的地址：

![](https://images.weserv.nl/?url=https%3A//pic2.zhimg.com/v2-f9f8096570148b85052a0331b271221c_r.jpg%3Fsource%3D1940ef5c)

这只是随手去网络上搜一下各地 IP 地址对标位置，就可以找到的信息。

所以显示准确的 IP 地址，才是真正的隐私。另一方面，这些精确的 IP 地址，是每一个 APP 都可以准确掌握的。也是司法部门很容易就可以掌握的。

通过 IP 地址将对方所在省份解析给你，和通过手机号码将对方所在省市解析给你，道理是一样的。当你接到一个陌生电话，然后显示这个电话来自哪里，你也不会觉得这是一种 “隐私”。

接受一条信息的时候，接受者可以反推这条信息，知道对方至少是哪里人写的、知道他大致的立场和意图，这使得收发信息的双方在信息上相对更平等一些，是一种安全机制。

我们都曾经有过被陌生人诱导情绪的经历。比如 zhyyyt 被网暴事件，现在竟然无法知道最初举报 zhyyyt 的人是谁，这就是隐匿身份的人诱导信息的一个事件。这种事情是不公平的。

在无数次的类似事件之后，行业自发或者被要求进行一次以显示发文者地理位置）精确到省份面对要求，是恰当的。

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-70377a3eb14cbed31673c3bdcd9e41e6_r.jpg%3Fsource%3D1940ef5c)

而这个简单打码的 IP 地址，你随意把它补全，去百度一搜，轻轻松松就能知道发帖人的归属地信息，甚至具体到了市：

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-690e9a31285a7aff0f597994bce6bf58_r.jpg%3Fsource%3D1940ef5c)

前面忘了说，这波牵扯到微博、微信、知乎、抖音、小红书在内的一众平台，所展示的 IP 归属地仅仅的到了国内省份或直辖市，国外国家或地区的程度。

这么看，明明过去所展示的信息比现在多，为什么这轮互联网「复古」却在舆论场上掀起那么大的波澜？今天就和大家好好聊聊这个事。

**一些担心**

在写这篇文章前，我找到了咱们之前做过的一个调查，即「社交平台该不该显示用户地理地址？」

投票结果如下：

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-f153d9c1359db51358bb7793e878f76a_r.jpg%3Fsource%3D1940ef5c)

有近一半的小伙伴选择了支持，还有一半的小伙伴持反对或其他意见，而我就是有其他意见里的一员。

为啥这么选，给大家说说我的想法——

1、有隐私泄露的风险。

如果说仅仅展示精确到省的 IP 归属地，那其实像 20 年前那般无伤大雅，但问题是我们所暴露在网上的信息远不止如此。

你随手分享的照片，随口说的话，甚至是在这些平台上和现实好友的互动，综合起来会还原一个大致的生活轨迹，而展示 IP 归属地无疑让有迹可循的生活被进一步的暴露在所有人眼前。

我知道，肯定会有小伙伴想说反正自己的生活轨迹也不重要，不怕暴露这么点的「隐私」，但怕不怕是一会事，受到骚扰是另一回事，微博全量现实 IP 归属地的那天，就有人遭到了同城私信的骚扰。

2、有助长地域黑的风险。

身为一个河南人，平常在网上没少见到一些地域黑的存在，诚然自己知道这些所言非真，但心情难免受到影响。

公开 IP 归属地后，碰到意见相左的情况，两个人的观点之争，在情绪的激化下，可能就会招来地图炮，这样的事情在微博的评论区并不少见。

3、有一刀切的风险。

这次社交平台上线 IP 归属地展示，并非是在敏感话题下才有，而是全量全内容的强制开启，且用户没有关闭的渠道。

但前面也说了，IP 归属地是可以「伪装」的，从过去的前台匿名，后台实名，变成现在这般前台展示，归属地也可能反过头成为恶人手里的武器。

......

**但我支持**

上面是在公开 IP 归属地后的几天里，我上网冲浪时观察到的一些现象，至于网上常说的防君子不防小人，或温水煮青蛙泄露隐私的论调，我觉得都有些偏颇。

看到这，我猜大家是不是都觉得我是反对公开展示 IP 归属地的？

不好意思，担心归担心，但我这次是真的支持，原因很简单，现在的互联网环境需要真实。

我不只一次的感慨，现在互联网环境相比十几年前愈发的逼仄，逼仄不是源于监管，而是源于情绪至上的二极论，扣帽子、打标签的行为随处可见。

诚然，这背后的原因很难归一，但我相信有平台监管不到位的问题，更有个人昧着良心赚流量。

就比如上个星期湖南长沙自建房倒塌的新闻下，有人冒充本地人留言，说自己刚刚经过事发地，甚至还在附近住过多年。

而当 IP 归属地显示后，却发现那个留言者留言时正身处陕西。

我想不明白这么做到底能收获什么呢，更多的评论？更多的点赞？我只知道，他的「无心」之举，为互联网带来了噪音。

像这样的噪音，热点下比比皆是，造假的成本极低，掀起的风浪却不小，更甚者恶意引导，嗓门大的反而能替代了沉默中的真相。

言论自由重要嘛？

作为有年头的网上冲浪先锋，我觉得重要，但慢慢的，我更认识到言论自由不等于无限的自由——

因为无限自由的背后，是无序。而无序导致的结果，在舆论公共空间内是乌烟瘴气，乱象丛生。

所以在当下，公开展示发言的 IP 所属地，是有必要的，这么做不是给官方看，给监管部门看，而是给我们这些用户看的。

公共讨论的空间自然需要公共监督，其实就这么简单一句话。  

我们需要看到真实，至少当我们区分消息真假时，能让造假者多一些成本，多一些辨识度。

权衡之下，利大于弊，是我支持公开 IP 所属地的原因。

**最后**

其实不管怎么样，显示 IP 归属地的时代真的又一次来了，只不过，这次不像 20 年前那般为了拉近网友们的距离，而是为了保持距离，还原那一点真相。

至于前面提到的 3 点担心怎么办？我想平台在净网的同时，也该扛起重任了——

曾经纵容的流量密码该不该扔？曾经缺乏的审核该不该严？这次显示 IP 归属地后该不该继续细化场景，完善隐患？

其实显示 IP 归属地只是开始而已。

什么是 IP 地址呢？

IP 协议中有一个非常重要的内容，那就是给[因特网](https://www.zhihu.com/search?q=%E5%9B%A0%E7%89%B9%E7%BD%91&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2483456319%7D)上的每台计算机和其它设备都规定了一个唯一的地址，叫做 “IP 地址”。

由于有这种唯一的地址，才保证了用户在连网的计算机上操作时，能够高效而且方便地从千千万万台计算机中选出自己所需的对象来。

IP 属地和 IP 地址有什么区别？

其实，用户 “IP 属地” 与“IP 地址”的概念、外延不同，混淆不得。就国内而言，前者指用户 IP 的归属地，即 IP 运营商所属省份，涉及用户数量庞大，少则几百万上千万人，多则近亿人；后者才是对应用户设备终端，专属个人所有，类似身份证中的家庭住址。

所以在网络上泄露用户的 IP 地址是不允许的。

**IP，网络世界的门牌号**

在现实中，人们用经度、纬度标记地理位置；在网络空间里，人们依靠的是 IP 地址。

![](https://images.weserv.nl/?url=https%3A//pic3.zhimg.com/v2-b936855a189035696cfc957c40624bc1_r.jpg%3Fsource%3D1940ef5c)

这是一个典型的 IPv4 地址，分为四个部分，共八个数字｜Wikipedia

IP 地址，就是 IP 协议所定义的地址。这里的 IP 协议，就是 TCP/IP 通信协议。

上世纪七八十年代，人们刚开始尝试网络连接时，互联网并不像今天，[万维网](https://www.zhihu.com/search?q=%E4%B8%87%E7%BB%B4%E7%BD%91&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2470875403%7D)（WWW）一家独大。那时出现了计算机科学研究网络、ALOHA 网、因时网、[阿帕网](https://www.zhihu.com/search?q=%E9%98%BF%E5%B8%95%E7%BD%91&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2470875403%7D)等不同类型的网络，彼此之间信息互通成了难题。

于是，有人开始研究[计算机网络](https://www.zhihu.com/search?q=%E8%AE%A1%E7%AE%97%E6%9C%BA%E7%BD%91%E7%BB%9C&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2470875403%7D)共同遵守的 “语言”。终于在 1978 年，斯坦福大学的教授 Vint Cerf 和项目经理 Rob Kahn，开发出了 TCP/IP 通信协议。这下，所有网络下的计算机就都能“对话” 了。

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-916ef79bf77cf821a47be4d8b83bf2eb_r.jpg%3Fsource%3D1940ef5c)

到了 1983 年，互联网前身阿帕网正式运行 TCP/IP。

其中，IP 确定了寻址方法、数据包的封装结构，最终让数据从源头主机传输到目的主机。人们经常把 IP 地址比作互联网中的门牌号，A 在这个屋子，只有在知道 B 的 “门牌号” 的前提下，才能去“串门”。

这时候，有人把自己的屋子改成了学校、商店等服务性场所，他们觉得八位纯数字的 IP 地址（IPv4 版本）不好记，于是发明了一套 “域名系统”——DNS。从此，域名（比如 [http://www.xxx.com](https://link.zhihu.com/?target=http%3A//www.xxx.com)）和 IP 地址（比如 192.168.x.x.）互相映射，访问哪个都能到达目的地。当然，每个人都会选择更容易记的域名。

可以说，互联网本质上就是一个 IP 地址对另一个 IP 地址的访问的总和。在这里，IP 地址是最基础的单位，每个联网的设备都有。

IP 地址往往是被分配的，某种意义上，用户并不拥有它。

![](https://images.weserv.nl/?url=https%3A//pic3.zhimg.com/v2-0dd937e8ec17732744b380a1e803b5f4_r.jpg%3Fsource%3D1940ef5c)

具体而言，当你使用家中宽带联网时，运营商会给这条宽带分配一个 IP 地址，连接在其上的任何设备，都同享一个 IP 地址。而当你是使用移动信号访问网络时，则由附近的[移动基站](https://www.zhihu.com/search?q=%E7%A7%BB%E5%8A%A8%E5%9F%BA%E7%AB%99&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2470875403%7D)分配 IP 地址，随着你的移动，你获得的基站 IP 也会不断变化。

此外，还有通过互联网数据中心（IDC）联网的方式，相较移动信号和宽带，IDC 机房使用的 IP 号段较为固定。

无论是运营商、基站还是数据中心，都有真实存在的地理归属，因此其发放的 IP 地址也如电话区号一般，标示出了这些属地。如 220.181.22.1 为电信在北京的、210.22.84.3 为联通在上海的。这些 IP 地址的属地，很容易搜索得知。

总结一下，在现实里，人们知道了你的门牌号，就可以直接找到你家去，但在网络空间里，知道了你的 IP 地址这个 “门牌号”，并不一定能找到这台设备的具体位置。因为 IP 是可以移动和变化的。

有一些网络使用习惯会导致 IP 归属地的变化，比如很多重视数据安全的公司，会要求员工在酒店等非加密网络环境下办公时，使用 “代理”，它相当于中途拦截了数据，让数据的目的地变为另一个地址后，再从那个地址传输给你。此外，使用公共 Wi-Fi 也可能会导致具体设备 IP 的隐藏。

**“暴露” 的信息，又多了一个**

人们在网上暴露的信息太多了，现在又多了 IP 属地这一个。

此次各网络平台公开 IP 属地，大家的担忧之一源于对隐私的侵犯。如一个流行的评论所称，从 IP 开始，之后是区域、街道、小区、楼栋号、门牌号、身份证，人们担忧隐私被一步步蚕食。

如果仅看 IP 地址，这个担心可能多余。即便知道具体的地址号，查询时也主要显示的是国家 / 地区 / 城市、经纬度、IP 主机名称、互联网服务供应商等信息。使用者姓名、准确地址、电话号码等可直接定位到个人的信息，仅靠 IP 地址难以获得。

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-f6f2c4a81a1a147196c8dd2638dc6c1c_r.jpg%3Fsource%3D1940ef5c)

北京网络行业协会法律专业委员会副主任王琮玮表示，目前 IP 归属地范围仅显示到省份或国家，没有直接指向，也不可能仅凭某省属的 IP 而关联到特定的个人。

但平台不仅仅掌握 IP 信息，它们获得的多种数据，依旧可以还原一个人的大部分信息。

以某主流平台为例，其用户协定中显示，其获取用户的信息包括用户身份与鉴权信息（如自然人身份、账号、基本资料）、使用过程信息（如位置、联系人）及设备属性信息。如果用户使用了特定服务，还会相应收集身份证号、面部识别特征、支付账号等更进一步的信息。

此外，个人在该平台发布的内容、好友关系、活动痕迹等，也属于平台信息采集范围。

社交平台通常是前台匿名，后台实名。2015 年开始施行的《[互联网用户账号名称信息管理规定](https://www.zhihu.com/search?q=%E4%BA%92%E8%81%94%E7%BD%91%E7%94%A8%E6%88%B7%E8%B4%A6%E5%8F%B7%E5%90%8D%E7%A7%B0%E4%BF%A1%E6%81%AF%E7%AE%A1%E7%90%86%E8%A7%84%E5%AE%9A&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2470875403%7D)》首次提出 “网络实名制”，即要求互联网信息服务提供者按照“后台实名、前台自愿” 的原则，要求互联网信息服务使用者通过真实身份信息认证后注册账号。中国科学技术大学公共事务学院教授左晓栋在接受[南方都市报](https://www.zhihu.com/search?q=%E5%8D%97%E6%96%B9%E9%83%BD%E5%B8%82%E6%8A%A5&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2470875403%7D)采访时表示：“由于我国实行网络实名制，每位网民都可能被精确追溯——这也意味着，倘若出现问题，可通过事后行政层面的监管手段来采取措施，而无需事前公开地理位置。”

实际上，除了隐私，人们担忧正是属地信息加上其他信息，会为显示 IP 属地增加其他含义。比如助长 “地域黑”、经由属地信息预设发言立场等。更何况此次公开 IP 属地并没有经过用户同意，可秉承的法律依据也未正式生效。

![](https://images.weserv.nl/?url=https%3A//pic4.zhimg.com/v2-7af9d788b50498452754cbe468dd0109_r.jpg%3Fsource%3D1940ef5c)

泄露隐私，对个人权益造成侵犯，从来都是综合所得。

**我该怎么保护自己**

辽宁昌信律师事务所[杨财广](https://www.zhihu.com/search?q=%E6%9D%A8%E8%B4%A2%E5%B9%BF&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2470875403%7D)律师在接受南方都市报采访时表示，IP 地址公开的信息，属于个人信息范畴，尚不涉及用户隐私权。

在网站论坛时代，人们无需登录就能发言，靠 IP 地址互相辨识。之前还有一款基于腾讯 QQ 的第三方辅助软件——[珊瑚虫](https://www.zhihu.com/search?q=%E7%8F%8A%E7%91%9A%E8%99%AB&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2470875403%7D)，它最大的特点之一就是获取对方好友的 IP 地址换算成上网区域。

但在今天，人们在互联网上的表达欲望和发布量级都不可同日而语。某种意义上，IP 并非是一个能完全独立出来的数据，它往往还和人们的生活信息有着强绑定。

在一些介绍 “人肉” 的教程中，心怀恶意之人只要用一个抓包工具，再给对方打一个电话，后者的 IP 信息就被 “钓鱼” 了。接着，他们就会用 IP 查询网站对 IP 进行大致的定位，然后再导入经纬度解析网站，等等…… 再到相关网站查询受害者注册过那些平台，继续在各个平台搜集对方信息。

相比 IP 暴露受害者的真实方位来说，人们有时分享出来的照片就带有精确位置信息。因为他们在手机拍照时，相机往往都是默认开启地理位置。

所以，在分享照片时隐藏位置信息，在设定里关闭 “精确位置” 选项，避免不怀好意的人仅从一张照片就得知了你的具体位置。

![](https://images.weserv.nl/?url=https%3A//pic2.zhimg.com/v2-1909c8670bc1bfdf2b72d6acfa2a9b5a_r.jpg%3Fsource%3D1940ef5c)

另外，在搜索引擎和网络平台上先 “人肉” 自己，留意自己哪些信息已经暴露，尝试删去不想暴露的信息；在各个平台上，可以用不同的用户名和个人资料，密码避免都设置成相同的格式和内容。

![](https://images.weserv.nl/?url=https%3A//pic2.zhimg.com/v2-66173c2d12877e52a4fe379a791f9181_r.jpg%3Fsource%3D1940ef5c)

最后，提防来源未知的软件，避免安装恶意软件，一旦被入侵，包括个人账号、密码、照片、购物记录、行程记录等等个人隐私都会被窃取。

黑客们会将泄露的用户数据整合分析、集中归档到一起，搭建 “社工库”，用此牟利。查询和搭建社工库是国家明令禁止的行为，情节严重要承担刑事责任。但目前有现象表明，社工库仍然活跃在互联网地下产业链。

回到本问题，为什么 IP 都只显示属地，而不是具体的 IP 地址：

1、IP 地址不具备直接可读性（不管是 IPV4, 还是 IPV6），非专业人员看不懂，看懂了也没有直接意义；

3、显示 IP 地址属地，告诉大家相关内容来自那里，而没有必要告诉大家它是谁，如果涉案追踪会有网警和相关专业单位负责；

4、APP 访问接口都会自带很多设备信息，相关信息公安机关在办案过程中是可以进行追踪查阅的。

自从抗生素被发现以来，人类一直在与细菌世界不断竞争，这些细菌随着新药的出现学会了快速生长。许多人认为，科学家能够在发现新疾病或病毒后立即研制出新药，但事实上，微小的害虫发展非常迅速，不断繁殖，并立即适应新的条件。

试想一下，一种普通微生物的寿命只有 20-30 分钟，而在这段时间内，它就能成功地留下了数量庞大的后代。

因此，无论这种药物有多强大，在数十亿种邪恶细菌中，总有一种可能会因为意外获得的突变而存活下来。一旦它传播开来，它将把这种能力传给后代。

通过这种方式，一种奇迹般存活下来的害虫很可能会在你体内养育出完全不被注意的整个后代。因此，无论我们说什么，我们每天都处于危险之中，因为我们永远面临着无法治愈的疾病，这些疾病就是那些在疯狂进化的微生物病毒。

火星上有生命吗？

一直以来，人们都对地球以外的世界感兴趣，尤其是我们的邻居火星。关于火星，人们有太多的看法，不过其中最引人争论的就是火星上的生命。那么火星上真的有生命吗？事实上，在[人造卫星](https://www.zhihu.com/search?q=%E4%BA%BA%E9%80%A0%E5%8D%AB%E6%98%9F&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2471211231%7D)和漫游者的帮助下，专家们在火星上发现了几个干涸的湖泊。诚然，它们里面的水并不像以前认为的那样新鲜，而是咸的。研究人员还发现，火星湖泊只出现在温暖的季节，随着寒冷天气的到来而消失。所有这些数据都表明，这颗红色行星曾经比今天温暖得多，其部分表面完全被[液态水](https://www.zhihu.com/search?q=%E6%B6%B2%E6%80%81%E6%B0%B4&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2471211231%7D)覆盖。而这个发现提高了火星曾经存在生命的可能性，甚至我们星球上的居民很有可能来自火星。

恐龙寿命更长

恐龙曾经是地球的霸主，几乎所有的教科书都说恐龙生活在很久很久以前。但几年前，一位著名的古生物学家的发现极大地改变了科学家们的想法。我们知道古恐龙在大约 6500 万年前灭绝了，但是在这位生物学家定期的挖掘中，意外地发现了三角龙化石的前部岩石。在研究过程中发现，这只恐龙的年龄大约是 3.4 万年，而不是 6500 万年。因此，史前人类很可能与恐龙并肩生活在地球上，如果这是真的，那么人类可能一直生活在恐惧中，毕竟恐龙的力量远不是人类所能比拟的。

头部移植

在几十年前，器官移植还被认为只是电视上的描述。但在今天，外科医生可以很容易地移植肝脏、肾脏、心脏，甚至手臂和腿。然而，直到最近，有人成功移植了头部。一位名叫塞尔吉奥 · 卡纳维罗的意大利神经外科医生进行了世界上最困难的手术，把一个人的头移植到另一个人的头上。

当然，这一大胆的举措让整个科学界兴奋不已，也让全人类兴奋不已。头部移植的主要问题是脊髓，因为它与颅骨中的其他重要组织直接相连，负责将所有必要的信号传输到身体的其他部位。如果它被损坏了，那么将根本无法感觉到任何东西。但这正是这些行动的全部意义，这些行动旨在帮助那些身体已经完全残废，必须完全替换的人。因此，如果这一极其复杂的操作成功，它将使世界科学不仅向前迈出一步，而且真正地向未来迈进一大步。

事实上，地球上的许多发现或者创造足以改变人类对地球的看法，甚至改变人类的未来，你认为在未来人类会怎么发展呢？这是一个值得深思的问题！

码农一枚，从网络角度上回复一下该问题：

首先目前使用比较普遍的还是 IPV4 地址，但是除去一些专用地址，网络设备真正可用的地址也才 36 亿个左右，这是全世界范围的数量。所以 IPV4 资源是十分紧缺的，换句话说，在 IPV4 时代，给每个网络设备都分配一个公网 IP 地址是一件很难做到的事情。再者，给一些家用的电脑，手机分配公网 IP 是没有多大必要的事情，只有在某些商用的场景下，才有分配公网 IP 的必要，比如云服务器。由于以上两条原因，运营商想到了两个办法：

第一个办法：动态分配 IP 地址，题主如此提问估计是把 IP 地址错误的认为是跟省份证号码或者手机号码一样，一个号码对应一个人。但事实并非如此，为了缓解 IP 分配压力，释放长时间没人用的 IP, 运营商每隔一段时间会重新分配 IP, 比如你第一次上网显示 IP 是 8.8.8.8，等你路由器重启之后，你的 IP 可能就变成里一个 IP 了，而你之前用的 IP 很可能就分配给别人了。所以这里 IP 和真实的人之间没有绝对对应关系，那展示 IP 的效果可想而知，除非某些公司用量专线网络，他们的 IP 是固定的，但是你知道 IP，还要去专门的备案网站查询是谁拥有这个 IP, 很明显这里显示 IP 并没有太大的意义。

第二种办法：小范围内用户共用同一个公网 IP，其实这种方式引用最广泛，在你的办公室，一般情况下只有一台路由器，那么连接在这个路由器下所有的设备的公网 IP，都是一样的。你可以做个测试，用你的手机和你同事的手机在百度搜索 IP, 会系那是你们的 IP 地址，你会看到你们的公网 IP 是相同的。运营商可以把这种范围扩大到一层楼，甚至是一栋楼，那么你们一栋楼的用户都使用一个公网 IP, 那么这时候显示公网 IP, 没有多大意义，因为一个 IP 后面对应了几十个甚至几百个用户。
