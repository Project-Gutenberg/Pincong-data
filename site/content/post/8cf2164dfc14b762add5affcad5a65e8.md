### 知乎用户  长夜未语冰​ 发表
    
在一个多数人主导集体的社会，不存在什么桶蘸价值，只有少数服从多数。

只有对集体的掌控权本就位于少数人手里，并且他们也认为自己真正代表的是少数人利益统治多数时，才会搞出桶蘸对象这个东西作为一种工具，去拉拢联合其它少数制衡多数。而所谓桶蘸价值，其实就是衡量工具的价值。

举个例子就明白了，在一个[专治社会](https://www.zhihu.com/search?q=%E4%B8%93%E6%B2%BB%E7%A4%BE%E4%BC%9A&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2522554507%7D)，少数掌握权力且代表自身利益的是 a，占人口绝大多数的主体是 b，少数不掌握权力的是 c

本质上 b 和 c 都是 a 防范的对象。但因为 c 是少数，其掌握的资源并不构成统治者维系统治的根本，闹得再大撑死也就是分离出去，并不能从根本上危及统治者的统治。而 c 本身又没有太大的剥削价值。因此在聪明的统治者看来，与其打击 c 不如将 c 拉拢收入麾下作为打手加以利用，把作为主体的 b 的一部分利益作为优待特权赋予 c，从而使得他们享受的特权优待与维持自身统治稳定高度绑定在一起。

于是乎，a 就是统治阶级，b 是被统治阶级，c 是桶蘸阶级

对 a 来说，如此一来减少了不稳定因素，而只要自己的统治不动摇，就能拥有一个永远忠诚且稳定效命于自己的中间阶层 c。如此何乐儿不为呢？

对 c 而言，自身利益很大程度来自 a 的供养而非生产，a 的统治稳定长久，就是确保其特权优待稳定长久的基础，因此会不遗余力地充当 a 的忠诚卫士。

至于 b，因为身为主体的他们拥有的土地、资源、创造的价值，是供养统治者维持统治的基本。所以被[统治者](https://www.zhihu.com/search?q=%E7%BB%9F%E6%B2%BB%E8%80%85&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2522554507%7D)视为打压防范的对象。一旦 b 产生自身作为一个整体的族群意识，那么立场就会翻转过来。统治者必须转而代表主体大众的利益，接受主体大众的选择；再也不能代表自身利益去统治压榨主体。

所以对于代表自身利益依靠压榨主体享乐的 a 而言，b 绝对不能产生普遍的族群意识，一定要让主体维持在分散、原子化、无组织化的程度，及时将主体产生集体意识的苗头扼杀在萌芽状态，否则自己的好日子就到头了。

这种统治方式总结起来就是以小制大，强枝弱干。统治者从被统治阶级中分化拉拢一部分少数派，赋予他们新的身份和特权并与自身利益绑定，从而形成统一战线进一步深化自身的统治力和影响力。再对普通多数进行打压、分散、去组织化，从而维持对多数统治的政治模式。

在正常情况下，a 是主人，b 是压榨的奶牛，c 是帮助主人监管奶牛的工作汪。

当然也不排除某些不识时务的 “工作汪” 不断作死挑战主人，硬生生把自己的桶蘸价值从正数作成了负数。毕竟工具再好，若不能安分为主人所用，甚至公然要和主人抢夺奶牛的利益分配。那么其地位和存在意义，就会从工具变成杀鸡儆猴的反面教材。对比现阶段还算安分老实的奶牛，必然会招致最猛烈的铁拳打击。于是在不明就里的旁观者看来，就形成了：“奶牛就是主人，所以代表自身的利益欺负可怜狗狗”的观感。

### 知乎用户 李晓宇​ 发表
    
看起来是按闹分配的逻辑，其实是对方实力的折现。

统战价值不是与生俱来的，它的背后是具备这一价值的人，其所在的群体能对占主导地位的一方进行阻挠，破坏的能力。这一能力可能是单向的，也可能是双向的。就是说，可能是要么为我所用，要么为敌所用，也可能是只要不为敌所用即可。

所谓统战，自然就是主导者让渡一部分自身的利益——如宣称、价值观、甚至金钱或者权力，来换取被统战一方的合作。这个行为的实际意义是，如果主导一方愿意消耗资源来替代被统战一方的合作，那么其损失会高于被让渡的利益，甚至有些合作方式是不可替代的。

而统战价值既可以产生，也可以消灭。因此被统战的一方为了不因误判而被判断为不再具备统战价值，反而会经常以各种方式来宣示。而对于主导一方来说，如果不出现力量对比发生突然的剧烈变化，现有的利益分配格局则是平衡的，维护这一格局对自身也有好处。

现行的对统战价值这一词的反感，主要是在主导方在践行让渡利益的时候，一方面越是地位低下，越对利益的损失（哪怕是很微小的）敏感，另一方面则是越是地位低下，越容易被让渡。可悲的是，这些涉及底层利益的变动，往往只是宏大叙事的注脚或者添头。

为了不触碰政治敏感，我们借用三国来举例。

[官渡之战](https://www.zhihu.com/search?q=%E5%AE%98%E6%B8%A1%E4%B9%8B%E6%88%98&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2521917072%7D)，许攸献策，曹操击败袁绍的故事大家耳熟能详。许攸初次见曹操的时候就甚为倨傲，而曹操也曲意迎合。很显然，曹操不迎合许攸是完全可以的，迎合意味着要让渡一部分利益。例如论功行赏的时候许攸要比别人分的多，那些心腹大将的想法啊，例如许攸如果之前曾经和曹操交战过，杀伤的曹营士兵的亲属的感情啊，例如自己的尊严和形象啊之类。

曹操认识到，如果打不赢官渡之战，自己怕是死无葬身之地，什么尊严形象将军士兵全都没有意义。因此他果断选择让渡了这部分利益，换取了许攸的合作。

后来呢，后来的事情大家也很清楚，许攸居功自傲，最后被杀掉了。许攸为什么会被杀掉？他是作为叛逃者来到[曹营](https://www.zhihu.com/search?q=%E6%9B%B9%E8%90%A5&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2521917072%7D)的，如果叛逃过去有可能被杀掉，岂不是影响恶劣？从这个角度看他的统战价值还在啊？
