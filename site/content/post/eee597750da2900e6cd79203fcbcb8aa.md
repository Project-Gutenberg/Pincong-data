同样是这个有高血压、糖尿病、肺心病的流感病人，中医在中药里怎么治疗？根据中医辩证，这个人是一个有多种疾病基础的病人，说明其本身体质差，属于正气不足，就是西医说的免疫力差，而且久病必然有淤有痰湿，就是西医说的体内代谢物沉积、血黏稠度高、[血管硬化](https://www.zhihu.com/search?q=%E8%A1%80%E7%AE%A1%E7%A1%AC%E5%8C%96&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1407384279%7D)、血脂高、血糖高，这样一个正气不足，体内多湿多淤的病人，感染了外界的病毒，属于正气不足，邪气侵犯，这样的病人正气虚弱，邪气盛，人体无法抗胜病毒，很容易因为这个流感引起多种基础疾病发作，因此，治疗就是首先要扶正气，然后是驱邪气，就是西医说的抗病毒。所以我们看中医的治疗，紧紧围绕防止疾病恶化，防止基础疾病加重，反而抗病毒本身不是很重要，治疗方案是：

补益气血 + 化痰祛湿 + 活血化瘀 + 驱邪（抗病毒）+ 止咳对症处理。

我们再一起看：

西医方案：抗病毒 + 抗菌 + 抗炎 + 止咳 + 化痰 + 降血糖 + 降血压 + 护心 + 吸氧 + 吸痰等。

中医方案：补益气血 + 化痰祛湿 + 活血化瘀 + 驱邪（抗病毒）+ 止咳对症处理。

这里，我们会明白一个中西医都知道的道理，药物可以有标准性，但是治疗方案是不可能统一的，没法标准化，因为每个人生同一个病，但是每个人又都不一样。

西医的双盲的虚伪性就明白的展现在我们面前，西医治疗不仅用了抗病毒药，而且还用了很多其他对症药，只是他们在统计的时候刻意忽略了。

例如，选一百个病人，其中的病人有糖尿病，高血压、冠心病、肺心病甚至肿瘤的。把这 100 个人打乱，让医生无法知道谁是试验对象，然后给每个病人开药，其中有 50 个用了抗病毒药，另外 50 个被另外的人换成安慰剂，通过治疗后，观察这 100 个病人中，使用了抗病毒药和未使用抗病毒药的效果，这个试验看起来很好，但是其实，我们仔细推敲，却很有问题。因为这 50 个患者除了使用抗病毒药物，他们同时也使用了其他药物，例如营养剂，例如降血压药，例如降血糖药，例如吸氧了，例如吸痰了。假设这些病人只使用抗病毒药物，这些人又有几个能治疗好呢？例如，如果有肺心病的，只给与抗病毒药，病人可能就因为[呼衰](https://www.zhihu.com/search?q=%E5%91%BC%E8%A1%B0&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1407384279%7D)、心衰直接死掉了。而且，那些没有使用抗病毒药物的，有些人也好了，这个怎么理解？

而西医某些人认为中医不科学，不能做双盲，他们认为，100 个病人，选 50 个用同一个方剂，另 50 个不用，然后观察治疗结果，结果发现这个方剂在这 50 个人里效果不好，经不起科学的检验。问题是，他们刻意隐瞒了一个问题。世界上没有绝对相同的 50 个人，50 个人里，有高血压的，有糖尿病的，有肺心病的，有营养不良的，每个人体质不一样，所以在中药的组合里，除了驱邪（抗病毒）成分可能一致，但是其他方面却都不一样，用相同的方子，治疗 50 个不同体质的人的流感，显然不可能都有效。糖尿病的流感病人，怎么可能和高血压的流感病人用一个中药方剂？两个人的体质就不一样，基础疾病不一样。而且，就算西医的处方也不一样，针对糖尿病的流感病人，西医是抗病毒 + 降血糖，而针对高血压的流感病人，西医的药物组合是抗病毒 + 降血压。

西医的双盲，把 50 个病人的治疗结果全部归功于抗病毒药物，实际上，这 50 个病人里，很多只要控制好高血压、控制好血糖、对症控制症状，不用抗病毒药，人家也能好。打个比方，这 50 个病人，同时都喝了王老吉，然后这 50 个病人就都是王老吉治疗好的吗？

其实要想中药做双盲，也可以，就是找 100 个体质一模一样的人，排除一切其他基础疾病和体质的干扰，但这也做不到，为什么呢，就算同一个人，疾病不同的阶段，他的身体内部病理破坏也是不一样的。就好比，早上血糖 16.0，到了晚上只有 4.0 了，这个时候，你还用同样剂量的胰岛素控制血糖吗？西医在治疗中，也在不断调整治疗药物和剂量，西医对于不同的病人，在不同的时期，使用药物方案也是完全不一样，为什么就要求中医使用一个方剂通治所有病人？这哪里是科学，完全一点不懂科学。

西医某些人的虚伪之处就在于混淆了治疗方案和治疗药物的区别。在西医的双盲里，因为抗病毒药是单独拿出来统计的，所以他们可以自由变换其他治疗药物，甚至可以自由调节抗病毒药物的剂量。但是中药呢，中药熬出来就是一大罐，全部成分都在里面，没办法在里面加什么药，减什么药，一旦一加一减，就变成另外一个方子了，这就不是一个方子治疗 50 个病人了。这就是为什么中医治病，同样是感冒，一百个人就有一百个方子，其实就是 100 个治疗方案。其实西医也一样，西医治疗一百个人，也是一百个治疗方案。西医某些人的虚伪之处就在于混淆了治疗方案和治疗药物的区别。

中医和西医其实都是治疗手段，取其长处，补自己的短处，这才是最科学的方案。

而且，西医某些人显然刻意隐瞒了人体自愈的重要性，对于大部分疾病来说，只要控制好症状，保证细胞营养物质吸收，保证供氧，把体内[代谢物](https://www.zhihu.com/search?q=%E4%BB%A3%E8%B0%A2%E7%89%A9&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1407384279%7D)清除干净，也就是中医说的，扶正气，驱邪，人体自然会往好的方向自愈。治病从来就不是某一味药的功劳，而是多种药物和人体自愈力一起作用的结果。就好比有句话，人要健康首先要细胞健康，而细胞要健康，离不开营养，离不开氧气，离不开内环境的干净，只要保证这些，大部分疾病，人体是可以自愈的。

西医治病效果 = 专科用药效果 + 对症支持药物效果 + 人体自愈力

中医治病效果 = 扶正气药物效果 + 驱邪药物效果 + 人体自愈力

治病，永远不要忘了人体自我修复能力，我们不是在治病，我们是在帮助人体自愈。

最后总结，中药方剂不是药，它是一个治疗方案。中成药也不是药，中成药是一种治疗方案，只是它适应的人群比较多，有普适性。同样，西医某些人，把整套治疗方案的功劳，全部归于某一味神药，这就是药厂卖药的伎俩，作为医生实在不应该这样。
