“这样吧，我赔给你 2000 块钱的[医药费](https://www.zhihu.com/search?q=%E5%8C%BB%E8%8D%AF%E8%B4%B9&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2664275473%7D)，你不要找公司闹了行不行？”

“不用……”

“你不答应？” 领导的语气很急。

“不是的领导，我的伤现在快好了，也没花什么钱，你只需把这两次的打车费给我报销一下就行了。”

“报，肯定报，欣欣没想到你这么善解人意，你确定不追求了？那我就跟总部人资说了？”

“嗯，大家本来就是一个公司的，天天抬头不见低头见的，将来还要领导多多关照我！”

“哈哈，一定一定！”

电话挂断，打开微信，领导迫不及待转过来 70 块钱，转账标注两天的打车费。

管峰感慨：“你怎么能这么快就原谅他，刚才他说给 2000 块你为啥不接着？”

我说：“接着，我就成诈骗了，犯不上，我就是想单纯的给他找点麻烦，吓唬吓唬他！未来起码三个月他都不敢再来招惹我。”

当然，我还是高估了他，这是后话，我们后面再讲。

回到 C 城后，我们重新坐下商谈，最终立了个对赌协议，2 年内，我们按照 9 折的价格订货，8 折设计承包，并授予一块合作单位的牌子，如果 2 年后，我们能把营业额做到两千万，不仅授予我们分公司的权限，还按照过去两年的订货和设计承包按照业务总额返给我们两成，以后供货和分包全部按照公司内部价格走。当然如果达不成，此前所谈的所有权益全部收回，并按之前订货以及设计分包的原价，进行差价补偿，后期的订货和设计分包全部遵照原价。

对此次谈判结果我们双方都非常满意。

我俩一出公司，就开心的在马路边又蹦又笑。

“恭喜管总，你离梦想又近了一步。” 我伸手。

“恭喜李总，你也离梦想又近了一步。” 握手。

管峰提议去吃大餐庆祝一下，我说一顿不行，起码得两顿。

回到 A 城后，我用我仅剩的两天休假到处找写字楼租办公室，最后我们在靠近商业区租了一套小别墅当办公室，一是价格便宜，二是小别墅带有一个小花园，可以修整一下，平时请客户来公司喝个茶谈谈业务。

拿到钥匙后，我们起草了一份股权协议，按照之前说好的，划分好股份比例，签上字，按上手印，从此我们正式成为合伙人关系。

回公司后，我继续上班，还是准点下班，领导再没找过我麻烦，隔了一段时间后，就有同事跟我一样开始准点下班，领导随表现出不满，但也没说什么。

我白天继续忙工作，偶尔跟着销售出去跑跑业务，积累一下销售经验，中午的时候约隔壁网络运营的同事一起吃饭，向他请教学习如何做广告投放，偶尔向新媒体的小伙伴了解一下新媒体的运营逻辑。晚上，回家编写广告，邀约客户，如有有意向，就让管峰第二天带着合同去拜访洽谈。

每天生活忙碌而充实。

两个月后，领导故态萌发，在下午四点的时候给我发邮件布置任务，我明确回复下班前完成不了，需第二日提交。领导装作没看见。

下班后，跟往常一样，关电脑走人，回家忙自己的小事业去了。

第二天，领导找到我们人资，指着我的鼻子骂：“这个员工不能要了，对工作毫无责任心，昨晚我明明告诉她这个工作有多么重要，她还是到点就下班，我们公司容不下这种毫无上进心的员工！我建议开除！”

人资万分尴尬，对他说：“欣欣编制不在我们这儿，她是总部派遣过来的管培生，我们地区没有权利开除。”

“那就跟总部打报告！” 领导气急败坏。

“可是您才是用人部门的领导，正常流程应该是您发起这次解雇流程。”

“我打就我打。” 领导甩手走人。

人资小姐姐尴尬的对我笑笑。

日子照旧，我还是准时下班，只是其他同事小伙伴被领导震慑住了，都不敢再准时走，每天羡慕的看着我准点走人的背影。

领导口口声声说要开除我，却没了下文。

后来还是人资的小姐姐告诉我，总部人资查了我过往的项目业绩，认为我业绩达标，不存在不负责任的行为，建议地区继续留用。

我差点笑出声，就喜欢你看不惯我，又干不掉我的样子。

一个月以后，我和管峰的业务单子多了起来，我正式递交了[辞职信](https://www.zhihu.com/search?q=%E8%BE%9E%E8%81%8C%E4%BF%A1&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2664275473%7D)。本来正常的话我需要提前一个月打申请，但我太清楚我们领导什么德性了。

果然，领导巴不得我赶紧滚蛋，递交完辞职信的当天，他就迫不及待的想让我滚蛋了，奈何我手里还有未完成的项目，他指定同部门的一个姐姐跟我做对接，让我当天交接完，当天滚蛋。

同部门的一个姐姐很是上火，她手忙脚乱的接收完我的工作，一面吐槽领导没人性，一边对我依依不舍。

“你辞职后打算干什么呀？” 姐姐问。

我眨眨眼睛笑道：“创业啊！”

辞职的第二天，我正式到我们的公司上岗，在这之前一直是管峰负责别墅的专修，我接到意向客户他就去洽谈，大单转包给 C 城公司，小单我们就自己画图做。俩个月时间，虽然忙碌，我们却小有所成。

我们开始招兵买马，我参考国内知名互联网公司，把新人分成 3 人一组，一人负责销售找订单，一人负责画图沟通，一人负责管理施工方进场施工，所得收货公司抽成一部分，大部分留给团队。

之前的同事有跑的过来打听，我给他们讲明公司规则，然后告诉我们是初创公司，给你们的薪资肯定没有原来公司高，但我们有高额奖金，还有每年的奖金池，当然，还一个你们最在意的，我们公司不加班，只要能完成当天工作，你随时可以下班，每个月能拿多少全看个人本事。

两年的时间，我们公司越做越大，逐渐发展成百来人的团队，我们从别墅搬出，正式搬进宽敞明亮的写字楼。

我们成了业内新贵，尤其是我们的壁画业务，其中给一个美术馆设计的壁画，竟然拿了个国际金奖，一下子打开了品牌知名度，给我们带来大量的订单。这期间，我利用大学导师的一些关系，揽到一个 2000 万的大工程，给一家五星级酒店做装修设计总承包，光这一个项目的业绩直接达成跟 C 城公司的赌约。

C 城公司给我们返还了两成的进货额和设计承包金，再给我们设立分公司的时候，我们拒绝了，我们保留了自己的品牌名字。为了表达对 C 城公司的一路扶持，也为了能更进一步获得 C 城公司在供应链上的优势，我开出 5000 万的价格出让公司 10% 的股份，让 C 城公司直接参股我们公司。

开庆功宴的当天，管峰官宣了他跟公司里的一个优秀小姐姐的恋情，我也牵起了在 C 城公司曾经帮助过我们的一个高管的手，亮出了手上的钻戒。这些年，我跟管峰因为共同的理想走在一起相互鼓励一同创业，又在创业期间纷纷找到了自己另一半幸福。

梦想、友谊、爱情，我们都得到了。

庆功宴第二天，我计划跟未婚夫一起出国度个假，临行前到公司再交代一下工作，前公司变成现公司的人资小姐姐，一脸神秘的递给我一份资料。看到上面的照片我有点意外，不禁挑了挑眉。

“要见吗？” 人资小姐姐问。

“见，为何不见！” 我最爱干这种打人脸的工作。

透过门框玻璃，我看到前领导跟人资小姐姐在打感情牌，回顾当年在公司里如何如何。

我推开门走进去，坐到他对面，弹一弹手中的简历，率先开口。

“先自我介绍一下，我是这家公司的总经理李欣欣，人资刚跟我说有个面试，下面请您介绍一下自己。”

“你…… 你是李欣欣？” 这种情况显然超出了他意料。

“是，你在来之前没有查这家公司的老板是谁吗？”

“你是老板？不是一个叫管峰的人。”

“哦，他是我们的设计总监，我的合伙人。”

他呐呐了两声，没说出话。可见他在前公司的人缘里并不好，前公司那么多人跳槽到我这儿，他竟全然不知道。

“好了，请开始你的自我介绍。”

“……”

接下来，他开始磕磕巴巴的说起了自我介绍，人资小姐姐，抿着唇差点笑出声，我在桌下轻轻踢了她一下，注意形象，我们可是正经面试。

我全程盯着他，不放过他脸上的任何一个丑态。

讲了三分钟，他终于讲不下去了，推给我一份他之前在前公司做过的项目案例。

我瞄一眼项目标题，我没记错的话，这个项目应该是之前跟我辞职做工作交接的小姐姐做的。

“这是你做的？”

“是……”

“好，我了解了，你回去等消息吧。“，然后转向人资，” 晶晶替我送送这位贾先生。”

“好的，李总”

“李欣欣，噢不，李总，请稍等，我这次面试……” 他推开椅子想追出来。

人资小姐姐岂能让他得逞？忙拦住他，信口开河道：“李总马上要去赶飞机，去国外参加一个重要会议，请您这边走。”

走出门，我随手把他的简历丢进了垃圾桶。白天摸鱼磨洋工，晚上加班卖表现，这种职场戏精只配待在垃圾桶里。
