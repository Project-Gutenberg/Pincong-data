**對加密貨幣一向採取強硬措施的中國政府，昨日又釋出了新一波的禁令。有鑒於加密貨幣市場近期波動劇烈，三大金融業協會共同發表關於「防範加密貨幣交易炒作風險」的公告，並強調進行相關業務有違法之虞，更將受取消協會會員資格等處分。**  
  
中國傳出重磅消息，昨（18）日三大金融業協會共同發表了聯合**聲明**，**強調已禁止任何涉及加密貨幣的相關業務**，同時警告投資人，加密貨幣市場為投機市場，呼籲投資人應樹立正確投資觀念，珍惜個人銀行帳戶。  
  
這篇聯合聲明由中國互聯網金融協會、中國銀行業協會，以及中國支付清算協會共同發表，管束範圍遍及國內所有金融產業，對中國金融市場發展無疑是一大重挫。  
  
禁令中更提到，因**加密貨幣「嚴重侵害人民群眾財產安全」，任何會員單位不得以加密貨幣定價、承保、交易，並要求各級金融機構及支付機構確實遵守，增強社會責任。**  
  

```
包括但不限於：為客戶提供虛擬貨幣登記、交易、清算、結算等服務；接受虛擬貨幣或將虛擬貨幣作為支付結算工具；  
開展虛擬貨幣與人民幣及外幣的兌換服務；開展虛擬貨幣的儲存、託管、抵押等業務；發行與虛擬貨幣相關的金融產品；  
將虛擬貨幣作為信託、基金等投資的投資標的等。
```

  
  
禁令中也要求協會加強對會員單位的自律監督，若發現違反相關監管規定，將依照自律規範對業者採取業內通報，執行暫停會員權利、取消會員資格等處分措施，並向金融管理部門報告，若有涉嫌違法犯罪，將直接移送公安機關。  
  
**相關議論**  
  
  
對於中國金融業封殺加密貨幣，丹麥投資銀行 Saxo Markets 亞太區執行長 Adam Reynolds 就在接受彭博社（BloombergQuint）[訪問]( "https://www.bloombergquint.com/global-economics/bitcoin-dips-to-lowest-since-february-amid-china-crypto-warning")時表示，自己對此感到毫不意外。  
  

```
中國資本管制很可能將因加密貨幣盛行而面臨挑戰，因此對加密貨幣祭出限制，對中國維持資本管理至關重要。  
對中國整府而言，唯一可以容忍的加密貨幣，就是他們所主導發行的數位人民幣。
```

  
  
而加密投資機構 FRNT Financial 創辦人 Stephane Ouellette 則認為，中國之所以會在此時此刻頒布加密貨幣禁令，或許和近日馬斯克（Elon Musk）的作為有關。  
  

```
馬斯克把加密貨幣市場搞得一團混亂，人們盯著他的推文操作加密貨幣，  
即便特斯拉（其實是馬斯克另一間公司 SpaceX）的火箭升空這樣無關的新聞，BTC 也出現了巨幅買壓，而這必須停止。
```
