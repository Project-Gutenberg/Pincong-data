**多个平台同日官宣：将显示账号 IP 属地**

4 月 15 日，今日头条和抖音相继发布公告称，为维护真实有序的讨论氛围，减少冒充热点事件当事人、恶意造谣、蹭流量等不良行为，拟在个人主页等位置展示账号 IP 属地。

![](https://images.weserv.nl/?url=https%3A//pic3.zhimg.com/v2-e3b81a215b9b2f1624f795d97cd8c10b_r.jpg%3Fsource%3D1940ef5c)

今日头条通过头条安全中心发布公告表示，为维护真实有序的讨论氛围，减少冒充热点事件当事人、恶意造谣、蹭流量等不良行为，拟在个人主页等位置展示账号 IP 属地。平台展示的账号 IP 属地为用户最近一次发文或评论时的网络位置，境内展示到省（区、市），境外展示到国家（地区）。账号 IP 属地以运营商提供信息为准，相关展示不支持手动开启或关闭。相关功能即日起在网页版进行测试，将根据测试情况逐步在其他场景全量上线。

抖音安全中心发布的公告也显示，相关功能在 4 月底进行测试，将根据测试情况逐步在其他场景全量上线。平台展示的账号 IP 属地为最近一次发文或评论时的网络位置，境内展示到省（区、市），境外展示到国家（地区）。账号 IP 属地以运营商提供信息为准，相关展示不支持手动开启或关闭。

![](https://images.weserv.nl/?url=https%3A//pic2.zhimg.com/v2-4a31959d534f872f20b4a23917342f7c_r.jpg%3Fsource%3D1940ef5c)

此消息发布后引发网友热议，“抖音账号将显示 IP 属地” 登上微博热搜第一。

支持者认为，此举 “很有必要”。反对者则认为，用户无法选择是否公开自己的 IP 地址。

4 月 15 日，快手也发布公告称，为维护良好社区秩序，防止部分网友在热点事件中出现蹭流量、传播不实信息、冒充当事人等干扰正常讨论的行为，快手拟在个人主页等位置展示账号 IP 归属地。平台展示的账号 IP 归属地为最近一次登录时的网络位置，境内展示到省（区、市），境外展示到国家（地区）。账号 IP 归属地以运营商提供信息为准，相关展示不支持手动开启或关闭。

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-1155f3e4c35f43a02a81553b62162ad6_r.jpg%3Fsource%3D1940ef5c)

15 日晚间，小红书发布公告称，为维护真实互动的社区氛围，防治冒充热点事件当地网友造谣传谣的行为，拟逐步开放 “个人主页展示账号 IP 属地” 功能。

![](https://images.weserv.nl/?url=https%3A//pic2.zhimg.com/v2-62b141fa1a558f186672074354f96d3e_r.jpg%3Fsource%3D1940ef5c)

小红书表示，该功能上线后，将在用户个人主页展示 IP 归属地域信息，境内展示到省 (区、市)，境外展示到国家 (地区)，信息以网络运营商数据为准。此外，该功能在测试阶段，不支持手动开启或关闭。

**微博个人资料页已新增 “IP 属地”**

今年 3 月，新浪微博就宣布上线 “用户个人资料页展示近期发帖所在地” 的功能。

根据 @[微博管理员](https://www.zhihu.com/search?q=%E5%BE%AE%E5%8D%9A%E7%AE%A1%E7%90%86%E5%91%98&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2441932791%7D) 发布的社区公告，由于俄乌局势和国内疫情等热点事件频发，引发网友高度关注讨论。站方在加强违规信息排查处置的同时，发现仍存在个别冒充当地网友发布和传播不实信息的行为，为更好维护传播秩序，站方在前期功能基础上，决定上线 “用户个人资料页展示近期发帖所在地” 的功能，展示用户最近一个月内最后一次发布博文或评论的 IP 归属地。

![](https://images.weserv.nl/?url=https%3A//pica.zhimg.com/v2-8f67f7f7abfe6615261589b76220f8c8_r.jpg%3Fsource%3D1940ef5c)

目前，打开微博用户的 “个人资料页面 - 个人信息” 一栏新增 “IP 属地” 信息，显示归属地省市区等内容，并注明“IP 属地以运营商信息为准，如有问题可咨询客服”。

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-cbc7337b40427620675b9d5f682da71a_r.jpg%3Fsource%3D1940ef5c)

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-683113d20cc8f47c5ef6fb8bb8b7f2b5_r.jpg%3Fsource%3D1940ef5c)

如果自行查看本人的资料页面则看不到 IP 属地，也无法更改。但如果使用他人账号来查看本人的微博资料，则可以看到 IP 属地。同时，关闭新浪微博访问位置权限后，依然显示 IP 属地。

据[南方都市报](https://www.zhihu.com/search?q=%E5%8D%97%E6%96%B9%E9%83%BD%E5%B8%82%E6%8A%A5&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2441933394%7D)报道，“显示用户的 IP 属地” 已在国家网信办曾征求意见的部门规章中有所体现。去年 10 月 26 日，国家网信办发布《互联网用户账号名称信息管理规定（征求意见稿）》（以下简称《征求意见稿》），对 2015 年施行的版本进行修订。

《征求意见稿》新增了有关 IP 属地的相关规定：“互联网用户账号服务平台应当以显著方式，在互联网用户账号信息页面展示账号 IP 地址属地信息。境内互联网用户账号 IP 地址属地信息需标注到省（区、市），境外账号 IP 地址属地信息需标注到国家（地区）。”

正常来讲，破坏一个平台要比建立一个平台容易得多，正所谓 “一颗老鼠屎，坏了一锅粥”。

一个正常的平台，就像一锅八宝粥，其中的人群确实像五谷杂粮，各种各样的都有。

水军和骗子这些群体，就是有伪装的老鼠屎，装作正常的 “五谷杂粮”，却始终在散发恶臭，给正常的“五谷杂粮” 带来非常差劲的使用体验，而且对整个锅和粥都有负面作用。

在有限的审核机制下，就容易出现 “劣币驱逐良币” 的现象。

" 劣币驱逐良币” 是经济学中一个古老的原理，它说的是铸币流通时代，在银和金同为本位货币的情况下，一国要为金币和银币之间规定价值比率，并按照这一比率无限制地自由买卖金银，金币和银币可以同时流通。

由于金和银本身的价值是变动的，这种金属货币本身价值的变动与两者兑换比率相对保持不变产生了 “劣币驱逐良币” 的现象，使复本位制无法实现。比如说当金和银的兑换比率是 1:15，当银由于银的开采成本降低而最后其价值降低时，人们就按上述比率用银兑换金，将其贮藏，最后使银充斥于货币流通，排斥了金。

如果相反即银的价值上升而金的价值降低，人们就会用金按上述比例兑换银，将银贮藏，流通中就只会是金币。这就是说，实际价值较高的 “良币” 渐渐为人们所贮存离开流通市场，使得实际价值较低的 “劣币” 充斥市场。

这一现象最早被英国的财政大臣格雷欣（1533-1603）所发现，故称之为 “格雷欣现象”，他观察：消费者保留储存成色高的货币（undebase money，贵金属含量高）——所谓“良币”，使用成色低的货币（debased money）——所谓“劣币” 进行市场交易、流通。

劣币驱逐良币的说法，现在逐渐用来表述，坏的把好的淘汰，劣币驱逐良币的原因，往往就是信息不对称和监管不够有力。

互联网平台上也存在 “劣币驱逐良币” 的情况。

劣币对应的是水军、骗子等群体，他们有手段肆无忌惮地低成本地用小号来引战、拱火、骂人。

良币对应的是比较遵守[公序良俗](https://www.zhihu.com/search?q=%E5%85%AC%E5%BA%8F%E8%89%AF%E4%BF%97&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2443009942%7D)的一群人，一不采用技术手段搞小号，二不愿意拉下脸皮对骂。

在监管不够有力和信息不对称的情况下，良币不确定对方是劣币还是良币，如果以对待良币的态度来对待劣币，就会收到意料外的暴击——扣帽子、泼脏水、辱骂等等。

如果处理不好，就会出现劣币驱逐良币。

劣币对良币的驱逐有两种，一是良币自行离开平台，二是良币摆烂变成劣币。

展示 IP 归属地，一方面提高了劣币的运营成本，另一方面，良币在选择摆烂时会增加一定的顾虑，可以一定程度上遏制互联网平台 “劣币驱逐良币” 的现象。
