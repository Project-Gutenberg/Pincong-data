研究一下这个表格，我们可以发现，人物的**基础属性**有这么几种：

*   CHR(Charm): 颜值
*   INT(Intelligence): 智力
*   STR(Strong): 体质
*   MNY(Money): 家境

事件的发生可能会带来颜值、智力、体质、家境的变化。

**生命的变化只有 “-1”**。包括 “你死了” 和“飞升天界”两种情况。所以修仙成功的朋友们不要太得意。你以为的位列仙班，没准就是被天道采摘成熟果实呢？๑乛◡乛๑

“**有某事件时才能被随机到**” 和 “**有某事件时一定随机不到**” 造就了命运迥异的农村线、城市线、美籍华人线。天赋（TLT, talent）对于事件的发生也有影响，如城间高楼必定出生在城市。

农村线我细看了…… 真的一言难尽…… 后面会说。

发生事件后，可能会根据前置事件和人物属性走向不同的分支。例如 “你遭到了一位老师的猥亵” 事件在人物 INT>7 时会走向 “你报了警，老师被刑拘” 反之则是“你逐渐消沉，但不敢和其他人说。”

出身这件小事儿
-------

然后咱们来聊聊命运迥异的农村线、城市线、美籍华人线。

我必须提前说，我是很敬佩这个游戏的开发者和文案的。我能感觉到那些过于现实的设定后，隐藏的悲悯之心。这个游戏根本不是用来当第一人称 “爽文” 体验的。它更像一部电影，是希望用户在第三人称观察的。

*   农村出生条件：MNY<8 或 天赋 (乡间微风)
*   城市出生条件：MNY>1 或 天赋 (天龙人，城中高楼)
*   美籍华人条件：MNY>8 或 天赋 (美籍华人)

也就是说只要初始家境 < 8 都有出生在农村的可能。但是不足 8 仍然可能出生在城市。

**智力、颜值、家境、体质等基础属性固然重要，但是出身影响可能发生的事件范围**。实践证明，如果某人家境为 3 但有幸出生在城市，仍然大概率比家境为 7 但出生在农村的过上更好的人生。农村出生很大概率遇到小混混打架、吸毒嫖娼、父母死亡等情节，多高的属性都得喝上一壶。在另一个极端，美籍华人线的正面事件比例极高，即如果使用美籍华人天赋，即便初始分配低家境，也基本不受影响。

**无奈地讲，之所以把快乐拿出来说，也是一种矬子里面拔高个了**。同样的高智力，帮助城市线小孩走向了发 paper 的科研人生，但在农村线小孩身上，好像只能止步于 “你发现你和其他人的想法好像都不一样” 而已了。

农村出身的出路是什么？上面的 “你去南方 / 北方 / 杭州 / 蒙古旅游” 成为**旅游达人**是种出路。

**另一个出路是运动员**：考上县城初中，每天要走很远去上学 → 被视察的体校教练发掘。去体校训练 → 进入省队 → 表现突出，进入国家队 → 刻苦训练 → 刻苦训练 → 参加奥运会 → 夺得金牌 → 并打破世界纪录 → 退役。 开始当教练。

**还有一个是电竞高手**：开始去网吧，沉迷游戏 → 游戏水平很高，远近网吧闻名。网吧老板允许你免费上网 → 被公司发掘，进入电竞培训队。前往大城市 → 连续夺得大奖 → 巅峰时期退役。 转行游戏主播。

其他的暂且留给大家发掘吧。

个人感觉最好的出路大概是：高考，你考上了专科 / 当地的普通本科 / 大城市的重点本科 → 你大学毕业后回村当了老师 / 护士 / 医生 → 在乡下工作的一个好处是，没有城市里那么繁忙 → 你的工资虽然不高，但这份工作你挺喜欢的 → 你现在很受村民们尊重。

如此最好。

生于城市
----

相比农村，生于城市简直是开启了人生的 Easy 模式。

说几个我发现的亮点。

### App 鄙视链

*   “你喜欢用父母的手机玩王者荣耀” 不限制农村城市。只需 MNY>2。
*   “你喜欢用父母的手机刷抖音” 不限制农村城市。只需 MNY>2。

然而……

*   “你经常在贴吧上吹水” 城市专属。且必须 MNY < 7。（贴吧：？？？）
*   “你经常玩 soul 和小红书” 城市专属。
*   “你经常用微博” 城市专属。
*   “你经常用知乎” 城市专属。
*   “你经常用豆瓣” 城市专属。

综上所述，农村孩子只能接触到王者和抖音。

### 娱乐圈入场券

在这个游戏里，走颜值线的巅峰就是少年 / 少女天团出道，然后进入北影，各种拍戏什么的。

这条路径具体是：你的美貌十分惊艳（条件：城市出身 + CHR>7）→ 你颜值出众，准备走娱乐路线 → 家人投钱帮你造势（MNY>7）→ 你参与选秀节目，一举爆红 → 你成为最热门的少年 / 少女天团 C 位 → 你被[北京电影学院](https://www.zhihu.com/search?q=%E5%8C%97%E4%BA%AC%E7%94%B5%E5%BD%B1%E5%AD%A6%E9%99%A2&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2106270191%7D)录取 → 努力的演艺生涯

农村线一开始就与此线无缘。城市线如果家境不足也会中途失败。

另外就是老生常谈的上学问题。

中国本土孩子的极限是清北（当然也可以通过留学走科研道路）。“你考上了世界名校” 为美华线专属剧情，甚至这个剧情对智力没有要求。如果 INT>8，则可触发具体校名如哈佛耶鲁[麻省理工](https://www.zhihu.com/search?q=%E9%BA%BB%E7%9C%81%E7%90%86%E5%B7%A5&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2106270191%7D)等。

嘲讽满分。

美华线虽然当起来人上人很爽，但也有它的贫乏之处。旅游，参军，电竞，出道，统统没有。唯一的成功路线就是成为科技精英。  
例如，你在美华线把颜值点满没有任何意义，因为不会有你的家庭给你在娱乐圈砸钱造势，因为亚裔的社会资源仍然很受限，亚裔的刻板印象就是没有魅力的书呆子。个人感觉这是一个有意的、精妙的设计。

注：评论区提到 “选了美籍之后把把被枪打” 的现象请检查**家境是否过低**。正常抽到美籍是提前保证 MNY>8 的，但如果用了天赋卡就不一定了。

这个时候按家境分流，同籍不同命，直面不同版本的美利坚。

![](https://images.weserv.nl/?url=https%3A//pic2.zhimg.com/v2-7ffef2b2e964530fb8599c410a4c8ee3_r.jpg%3Fsource%3D1940ef5c)

如果 MNY<4 随机到了 “你上不起幼儿园，父母开始教你学习” 这个事件，后续就可能出现诸如 “你平时很少出门，因为父母说外面很危险”、“你的父亲在地铁上被暴徒攻击，受了伤”、“班上的几个非裔同学经常欺负你” 等等危险事件，最终可能导致（尤其是你点了挑衅天赋的时候）你被枪打死或者被流浪汉群殴致死。

由于大多数人都是凭借初始分配 MNY>8 高家境抽到的美籍华人，因此并没有体验过底层美华不为人知的人生，美华线也就因此被当成人生巅峰的代名词了。其实如果初始分配 MNY>8 的高家境，在城市也会大概率过的很好。

一纸国籍可能值钱，也可能不值钱。人有钱它就值钱，人没钱它就不值钱。

* * *

其实这节标题里的 “朴实无华且枯燥” 不止是指（家境足够）美籍华人线。也指会在总评分数爆炸高的修仙线。

有人说修仙过程太枯燥，希望开发者多加点剧情。我倒是觉得，保持现在这种修仙的枯燥，在意境上反而可能会更胜一筹。同样的感觉也会在玩开挂版的游戏时产生，单调，枯燥，乃至虚无。完美的高分人生好像只由数字的叠加组成。

这种虚无感的另一个来源在于，此游戏是没有主动选项的。但真实的人生又何尝不是呢？**当一个受到周边不良环境影响的农村孩子，误入歧途去嫖娼赌博的时候，他会有一个 “按钮” 去“选择”吗？？！**

所有的创作，都有话想说
-----------

我在文章的开头说，我是很敬佩这个游戏的开发者和文案的。我能感觉到那些过于现实的设定背后，隐藏的悲悯之心。

这个游戏根本不是用来当第一人称 “爽文” 体验的。它更像一部电影，是希望用户在第三人称观察的。那它想和我们说什么？有些人说，作者纯属鼓吹国外月亮圆，所以才把美华线设定的那么顺风顺水，中国人，尤其农村，就活该受罪。

**并不是这样的。**

### 知乎用户 米特​​ 发表
    
先说结论：特色文案和易于上手的优点非常突出，直接引爆了玩家对于游戏的传播，在数值 / 成长 / 传播引导方面有改进空间，作为短期的二人团队小作品来说已经足够好了，下面是详细内容：

**优点**
------

**1，文案特色**

这样一个超 mini 体量的游戏，基本不存在玩法和逻辑创新的可能，唯一的创新可能在于内容设计，本作的文案内容就非常对用户胃口，使用了大量的网络梗

![](https://images.weserv.nl/?url=https%3A//pic3.zhimg.com/v2-04b41981a2b084d11f138714da1fdd53_r.jpg%3Fsource%3D1940ef5c)

而且梗和梗之前的类型非常广，有二次元梗，有时事梗，有[克苏鲁](https://www.zhihu.com/search?q=%E5%85%8B%E8%8B%8F%E9%B2%81&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2102648480%7D)，还有很多小众的梗以及伏笔，这样做的好处一是让玩家觉得贴近生活，二是产生了分享的主要动力，每一次玩家发现新梗都是一次分享的机会

![](https://images.weserv.nl/?url=https%3A//pic4.zhimg.com/v2-a0e5227df96459e4bd172f1599af7045_r.jpg%3Fsource%3D1940ef5c)

**2，简单与深度**

对两人制作小队来说，易于上手这点有天然的优势，只要选了特质就可以开始游戏，单局时间短意味玩家再来一局的成本低

![](https://images.weserv.nl/?url=https%3A//pic2.zhimg.com/v2-a132f23ca75210f1b0eba888ab055af5_r.jpg%3Fsource%3D1940ef5c)

而游戏丰富的随机性一是让玩家感受到世事无常（各种死亡），即使中途结束了游戏，玩家也倾向于认为是自己加点有问题或者运气不好而选择再来一局。二是通过尽可能少的开发量让游戏有更多重复可玩性，这种高性价比的思路也是众多 “再来一局式游戏的选择”

除此以外，在主要人生线路之外还设计了魔法少女路线和修仙路线，魔法少女路线是选择魔法棒特质 + 12 岁以前开始看动漫，修仙路线是选择小盒子特质 + 活到 100 岁，两种额外路线都极大的增加了游戏的耐玩度

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-2bd0cb6cede0fcf1781f56aecb9b8169_r.jpg%3Fsource%3D1940ef5c)

**缺点 & 改进**
-----------

说完优点后来谈谈游戏的缺点和改进，不像优点更偏体验，缺点部分会更偏游戏设计

**1，数值**

游戏的数值最终还是要为游戏体验所服务，不能脱离游戏的体验设计目标。

本作中橙色品质特性的刷出概率仅为千分之一，这个数值的设计目的可以理解，是为了让玩家重复更多次才能刷出橙色特质，增加游戏的重复可玩性

但是这种数值小于玩家可感知的范围，从数值上来讲属于无效设计

改进：应调高几率或通过其他方式给予玩家感知

![](https://images.weserv.nl/?url=https%3A//pic3.zhimg.com/v2-0ce61d8b49253d7404a96a5683191a30_r.jpg%3Fsource%3D1940ef5c)

**2，成长**

和数值一样，成长也应该为了体验而服务，本作作为短单局游戏，最重要的就是单局的重复体验性

而本作虽然成长范围很广（特质分为白蓝紫橙，还有属性）但全部投在了随机上，成长上仅能保留上局的一个特质，玩家成长弱，刨除猎奇以外重复游戏的动力衰减很快

改进：可以把部分随机的特质和属性改为成长，让玩家中后期重复游戏的动力衰减变弱，也可以划分更明显的成长阶段反馈（目前仅有活过 100 岁和 500 岁渡劫）

**3，传播引导**

作为小成本免费游戏，本作没有营收的压力，除了评价外，最应注重的就是裂变式传播效果。

本作没有对传播做额外引导，可能是出于制作成本或者制作意愿的考量

改进：不动游戏主流程，只加一个简单的保存结果截图分享对于传播效果的影响都将极为明显

整体来说，本作以极小的成本做出了传播性极佳的作品，证明在美轮美奂的美术效果和复杂的游戏世界外，有趣的核心玩法对于玩家的吸引力是无与伦比且性价比很高的，也证明重复可玩性的设计价值有多高

这个界面设计，椰树椰汁老板看了直呼内行，给你 10 分钟，相信你一定可以用 Word 或者 PowerPoint 做出一模一样的界面来。

再看一下文章开头游戏界面实际的样子，不能说一模一样，只能说毫无区别，这么高的还原度，各位产品经理估计已经感动到流泪了吧……  

就是这样一个粗制滥造的 “页游”，在短短几天内获得了超过 **400 万的玩家和 2 亿的访问量**。

靠的就是它有 “有趣的灵魂”，开局就是喜闻乐见的 “十连抽”（这游戏完全免费而且没有广告），你要从 10 个随机的天赋里选择 3 个，其中既有稀有度超高的强大天赋，也有完全负面效果的搞笑天赋，从天赋的设计开始，作者就已经开始疯狂玩梗了。

作为一个游戏玩家，比起社会现象，我更想探讨的是，文案、故事对于一款好游戏的重要性。

没有画面，没有配乐，连 4399 基本的互动性都做不到，但就是寥寥几个选项，却道尽了现实的一切。

疯狂，浮想联翩，却又如此真实。

是没法想象到，在这文案的背后，制作者对生活拥有多少的体验。他的目光所及之处，让我们做出的每个选择，都是无数人真实的一生。而同时，也道出了最赤裸残忍的事实：家境的确决定人生的上限。

本人亲测，用着最原始的版本（20 初始属性），通过极端的分配（最高为 10，最低为 0），验证出这样的结论。刚开始我认为，智力是最重要的。因为知识决定人的认知，也是最能跨越阶级的有利武器。所以出了 10 智力 + 10 体质 + 0 家境。但结果很遗憾，2 岁就被饿死了。

**选择，随机，接受所有意外，并享受它们。**

**因为我们的选择，大多都那么『顺理成章』。**

《人生重开模拟器》总共只有两位开发者，负责程序的小鸟和负责文案策划的小德。制作这个游戏之前，他们并没有做过什么宏大的设想和铺垫，只是想和群友们一起捣鼓出个有趣的玩意，用以自娱自乐。作为一名拥有传播学教育背景的开发者，小德平常对互联网上的各类信息资讯都比较敏感，持续保持着阅读和收集的习惯。小德在多个贴吧都已经达到了 15 级的夸张级别，还曾经是炉石 DIY 吧的吧务，并且贡献了大量的 DIY 作品。

由于没有在乡村生活的经验，出于严谨起见，小德在编写某些事件时还特意去请教了一下乡村出身的群友，从他们那儿获得一些参考。

**3**

《人生重开模拟器》的成功不是偶然的，可以说是两位开发者厚积薄发和群友们的集思广益所带来的结果。但是，《人生重开模拟器》的爆红，在给他们带来喜悦的同时，也带来了不少的困扰。作为一款本用以小圈子内自娱自乐的游戏，在编写《人生重开模拟器》的文案时，小德比较放飞自我，对很多东西进行了直接的讽刺，但在游戏爆火之后，就有各路人马找上门来，开始对他们指指点点甚至恶语相向。

在互联网如此发达的今天，一个火爆的作品自然会受到各种五花八门的审视，就算网上只有 0.1% 的极端分子，但是在如此庞大的网民数量加持下，他们也拥有着一支不可小觑的队伍。

玩家被抛入了游戏世界，一旦决定了最初的出身和禀赋，除了让离散的年继续流驶之外，没有别的干预手段。正是这种消极——无作为——躺平，构成了游戏的互动机制。很显然，在操作的层面上，这是无互动。

互动不在键鼠之上，而存在于空气之中。这是说，这种贫乏、无聊承接了对所在之世的模拟，这令人想到了学生预测领域的早期研究。我们在成绩预测场景的科研工作，约定俗成有这样的工作模式：

独立的[特征集](https://www.zhihu.com/search?q=%E7%89%B9%E5%BE%81%E9%9B%86&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2150681734%7D)通常用于不同的预测任务，例如，学生活动日志用于预测课程表现，注册数据用于预测长期的大学成绩。[\[1\]]()

什么是注册数据呢？是人口学变量，所谓的年龄、性别、种族、教育、工作状况和残疾等等，或者说对于一个大学生活的过程来讲，这是先定的。我们可以采取这种消极的断言：长期的成绩表现从这些人口学变量中就可以预测了。如果把这个过程放宽到人类寿命的宽度，我们似乎也可以采取这种消极的断言：人生的成绩表现从先定人口学变量中就可以预测了。

二、
--

然而，人生重开——或许可以说是诞生于神奈川的梗的包藏隐晦的社会批判的小品——并没有如同一个在本体论意义上消极的游戏那样，惟能带给人空洞的快乐和疲乏的享受。作为余暇的游戏之作，QQ 群伊始的二人小创意，它有八分不忮不求的底色。它是我们时代的民间文学——我们很难想象，一个大厂的游戏策划创作了这样的作品。亦或，一个幻想家能写出[圣杯战争](https://www.zhihu.com/search?q=%E5%9C%A3%E6%9D%AF%E6%88%98%E4%BA%89&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2150681734%7D)形形色色的怪人，他们有着异乎寻常的执念，但这种奇妙的想象未必能够触及到乡间微风的田园的娃娃亲、彩礼和恶霸的手眼通天，未必能够区分出美籍华人、天龙人和其他人。

这是这个游戏的严肃性。在无尽的十连抽里刷天赋，则更有[贝克特](https://www.zhihu.com/search?q=%E8%B4%9D%E5%85%8B%E7%89%B9&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2150681734%7D)的等待戈多之感：

爱斯特拉冈: 算了，我们走吧！  
弗拉季米尔：我们不能。  
爱斯特拉冈：为什麽不能？  
弗拉季米尔：我们在等待戈多。  
爱斯特拉冈：唉！

或者说，

【绝密消息】科学家认为大多数人的天赋都是来人间凑数的。

三、
--

如果把重开作为一种等待，或者重复，那么一种思就会变得盲目，盲目于它的时间交错的实质。每一个离散的年的事件，当它们单向地运动到尽头，就会折返到最初的起点，再因循一种疲惫的轨迹。在每一个这样的过程，能让人心安的唯有一种确凿的同构性，那是：

“我心想，一个人可以成为别人的仇敌，成为别人一个时期的仇敌，但不能成为一个地区、萤火虫、字句、花园、水流和风的仇敌。

然而它没有回答全部问题。这个人可以成为 “一个时期” 的仇敌吗？为了这个时期尽速过去，为了远离甚至杀死这个时期？没有抽取到好的天赋，一个人会成为这段旅程的仇敌吗？当然，重开游戏的代价是最弱最微的，所以没有友与仇的痛切之言。我们所知道的，是当这个过程往复了几十遍，自然就产生了一种序结构，它标明了什么样的人生是值得阅读的，而什么样的是不值得阅读的。

四、
--

在[博尔赫斯](https://www.zhihu.com/search?q=%E5%8D%9A%E5%B0%94%E8%B5%AB%E6%96%AF&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2150681734%7D)的名作里，因缘最古是追溯到虚幻的异域的先祖，为斯蒂芬 · 艾伯特所知的[彭冣](https://www.zhihu.com/search?q=%E5%BD%AD%E5%86%A3&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2150681734%7D)。因缘最近是追溯到一战，西线战场上的[艾伯特](https://www.zhihu.com/search?q=%E8%89%BE%E4%BC%AF%E7%89%B9&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2150681734%7D)。凭着姓氏和姓名的错合，不同的 spatio-temporal 结构糅合到了一起，这就是迷宫文学。

如果说 “重开” 作为一种戏言有什么让听众无法满足的，就是它作为一个迷宫却没有迷宫的影子。一个重开的人只有它自己，而这个自己的形态也是虚无。和这种虚无所对照的实有，正是某些竞技形态的网游，为我们所熟知的——不见得合适作为所谓严肃体验而提及的——LOL、[永劫无间](https://www.zhihu.com/search?q=%E6%B0%B8%E5%8A%AB%E6%97%A0%E9%97%B4&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2150681734%7D)等大众游戏。LOL 最早有召唤师的设定，后来则渐渐淡而不提，因为当一个玩家操作一个英雄的时候，它的技术、意识和策略定义了它自己。或者说，在这里是一种介入的可能性让人沉迷。

而在一种博尔赫斯主义里，可能性也被先定的宿命打败，也就是这样的洋洋大观：

我将小径分叉的花园留诸若干后世。（并非所有后世）

然而，这寄予着大观的若干后世，又意味着何种的分支结构，被何种的先定论束缚，这就是近来——这一系列的策划探究——所要直面的内容。它是以 TOUHOU 为背景，但更多是一次 egoistic 的自为探询。

对游戏好奇而去探索各种可能，这是因为站在了第三人称，如果不小心每次都以第一人称带入，玩下来只觉得绝望。这种绝望来自于，**这些样板式的人生和现实确有着绝妙的呼应。**

绝望一：要跨过坏境、天赋的限制的被观测难度。游戏当然是编写好的难度，它和现实不一样，真实的我们还有未知，既然未知，一切都还有希望。但当如果意识到大部分人的现实——奋斗一生后回首看自己的人生，发现自己从没有跳脱出大数统计的规律（出身，健康，‘天赋’等等）。人或许引以为豪的意识，在第三人称看来是大数分布，其中的奇迹就是奇迹，很残酷的，只是少部分人。**原本每个人都可以放在自己的未知上的奇迹和希望，突然变得不可相信**——这是绝望之一，预测本身。

我很喜欢前段时间流行的[人生重开模拟器](https://www.zhihu.com/search?q=%E4%BA%BA%E7%94%9F%E9%87%8D%E5%BC%80%E6%A8%A1%E6%8B%9F%E5%99%A8&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2262111306%7D)。世界很大，有无数我未曾见过的人生。所有的可能空间，在我出生的一瞬间就塌缩的极小，并随着时钟变得越来越小。我很想有机会看见那些未曾成为现实的可能性，与最灿烂的那些可能相比，最卑微的那些同样打动我。如这个简短的故事：“你出生了，是个女孩”、“你从小生活在农村”、“你父母又生了个儿子”、“你的家庭更加困难、吃不饱饭”、“你死了”。

当然，我可以用看书、生活、体验、实践来理解人生的模型，可是这些都太复杂了。只有电子游戏，才敢于给出精确的、简化的模型，毕竟，这只是游戏，何必那么认真呢？毕竟这是一段程序，不精确不简化还怎么写啊？与严肃著作相比，《人生重开模拟器》、《模拟人生》、《文明》等游戏，最无畏、有趣而可贵的地方就在于此了。

说回《人生重开模拟器》，我喜欢他，但我不愿去玩他。因为我实在等不了一次次漫长的、大同小异的随机过程。我自己的生活已经够真实了，在游戏里面为何不能做一回上帝？

所以，为了 “从游戏中体验人生 “这个可笑的目的，我忍不住看了游戏的源码，又写了一个 demo 帮助我阅读游戏的核心数据。有兴趣的朋友可以去看看我的

[demo](https://link.zhihu.com/?target=https%3A//share.streamlit.io/anselmwang/life_restart_analysis/main/main.py)

。

这个游戏给出了什么样的人生模型呢？简单的总结是： - 大部分人在重复相似的人生，每个年龄自有该发生的事情，时间的洪流将我们卷向未来。 - 到这里为止，我感到：人生而不平等，属性与天赋，是人生的随机数种子。 - 人生的每一年，并不是条件独立的采样过程，而是具有长程依赖关系的随机过程。

下面是更详细的解释。 首先，在每个岁数有一个事件列表，不同的事件有不同的权重。在每个岁数，游戏根据权重随机的选择一个事件。比如在 31 岁，幸运和不幸都不常见（0.03），平凡而普通才是正常生活

**每一局游戏，同一个年龄，不管是谁，都共享相同的事件列表，这告诉我：大部分人在重复相似的人生，每个年龄自有该发生的事情，时间的洪流将我们卷向未来。**

那么，如何体现个体差异呢？在游戏的开始，每个人有 20 点，可以分配在不同的属性上，每个人还有不同的天赋。这里简便起见，只介绍属性。可以控制的属性有下面这些：

```
CHR: "CHR", // 颜值 charm CHR
    INT: "INT", // 智力 intelligence INT
    STR: "STR", // 体质 strength STR
    MNY: "MNY", // 家境 money MNY

```

下面两个属性大家都一样：

```
LIF: "LIF", // 生命 life LIFE，初始时1
    SPR: "SPR", // 快乐 spirit SPR，初始有值

```

在游戏的过程中，不同的事件会改变属性，而属性又作为事件的前提条件，显式的 enable/disable 某些事件。如智力太低就很难触发 “中考考得很好，上了城里的好高中” 这个事件。这是随机的人生中，相对有脉络可循的部分。

**到这里为止，我感到：人生而不平等，属性与天赋，是人生的[随机数](https://www.zhihu.com/search?q=%E9%9A%8F%E6%9C%BA%E6%95%B0&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2262111306%7D)种子。**

除了属性外，一个事件的发生也会影响未来的事件。下面是我在 demo 中发现的有趣关系。 比如下图，“你进入流水线工厂工作”的一个前提 (观察 include) 事件是 “高考，你考上了专科”，而该事件的前提是：智力 < 6 并且发生过事件” 中考考的一般，上了县里的高中”。而当你考上一般高中时，无数的可能性对你关闭了（观察 exclude 部分），你不会“辍学打工，补贴家用”

另一个例子如下，大家自己看看 “你成为了世界首富” 的可行路径吧。另外，effect\_dict 里面指出，成为世界首富，会给你增加 1 点快乐(SPR)。

**在游戏的人生模型中，人生的每一年，并不是条件独立的采样过程，而是具有长程依赖关系的随机过程。**

### 知乎用户 Simon Xu​ 发表
    
在我个人看来，它其实算不上是一个游戏，甚至连小品级都算不上，最多算一个框架。

不要太过于捧杀了，当年的《太吾绘卷》，如今也是口碑反噬。

这个框架，连二次开发的价值都不是太大，因为结构太过简陋，不过是在一堆的条件假设框架下随机取舍推演而已，真要做成一个成品游戏，要解决的问题太多了。远远多于这个框架本身的价值。

但这并不是说这个东西就毫无意义。

它在市场探索上进行了一个很有益的尝试。因为它是一种全新的游戏类型。我姑且先把它叫做 “低投入度的观赏性游戏”。也就是说和市面上大多数的游戏不同，它没什么操作，玩家不需要费心思，只需要为人物设定一些先制条件因素，便可以观察人物在某些特定环境条件下的自我发展。这是一种上帝视角，它甚至不需要丰富的画面和动听的音效，只凭剧本推演就可以很有娱乐性。

这一点上，它和那些 “自动寻路[自动打怪](https://www.zhihu.com/search?q=%E8%87%AA%E5%8A%A8%E6%89%93%E6%80%AA&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2142374482%7D)” 之类的游戏还是有不同之处的，因为它注重的是剧情推演。

这其实是非常适合一些上班族的游戏，因为很放松，休闲，也很娱乐。所以这种游戏类型是有商业前景的。

先说说这东西的缺点：剧本单一，内容单调，其实刷个几十遍也就差不多了。而随机性的因素过多，也就是在推演上不可控因素过多，某种角度上增加了娱乐性，但降低了策略性和游戏粘度。

所以如果真要二创的话，肯定要冲解决这些缺点而来。

目前来看有几个方向，比如有一个方向就是走 “无限流小说” 方向。每次提供一个设计好的剧本，将时间缩短，将事件密集，将前后关联因素做好。然后将一个玩家设计好的具备不同心性，外貌，武力等因素的角色，丢到诸如《生化危机》之类的紧急场景中，观察它的生存和选择。使剧本环境因素可控和定量，让玩家可以逐渐熟悉和了解关键要素，然后让其可以反复尝试，增强策略因素。这就具备一个好游戏的条件了。

然后每周或每月定期出一个剧本，靠卖剧本赚钱就可以了。

这个游戏的开发没有什么难度，关键在于大量的剧本推演和文字功夫，这才是水磨功夫的地方。
