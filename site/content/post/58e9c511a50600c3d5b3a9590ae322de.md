4\. 恶意和难度问题
-----------

说实话这游戏在类魂里算是比较简单的，但每个人对难度的体感方差会非常大。这游戏的不同 build 强度差异巨大，轮椅非常多。如果你硬着头皮用很弱 / 被克制的 build 打，那这游戏会非常难，但你换个 build 可能拿脚都能过。但据我观察，大部分玩家是懒得换 build 的，所以每个人的难度体验肯定会差很多。

恶意 / 篝火的设置也是同样的问题。即使是现在这个水准的恶意，你是可以做到重开一周目从章节开头不坐火（甚至不只是一命）一直打到章节结束的（梦回魂 2）。如果没有恶意，对这个游戏强度靠前的 build 来说推图实在是过于简单了（当然对靠后的 build 还是很难）。篝火的设置也是同理，如果你没有续航你才需要很多的篝火。实际上在第二章获得杀怪回血的玉佩以后玩家的续航是很强的。当然，很有可能一大部分玩家并没有发现这个玉佩或者还没玩到这里就差评退款了。365 里路也是一样，如果你的 build 足以初见就能过 boss 就没有 365 里路的问题，只死几次也可以接受，只有你一遍遍被 boss 折磨的时候才会觉得绝望。

无可否认的是，这些设计都非常陈旧了，连老贼自己都学会了在 boss 门口放玛莉卡楔石，也不会如此大面积地滥用恶意。[夏思源](https://zhida.zhihu.com/search?content_id=740262313&content_type=Answer&match_order=1&q=%E5%A4%8F%E6%80%9D%E6%BA%90&zhida_source=entity)还抱着老一套不放，并且拒绝试玩 up 提出的减少恶意的反馈，也无怪乎很多玩家不买账。这又回到了第一点，也就是说，这个游戏这么设计的时候就注定了他的受众只是玩家的一小部分，你拿去卖给不喜欢你的设计思路的玩家，那大家当然会给你差评。

我个人并不太反感这些地图上的设计，甚至支持继续删除 20%-50% 不必要的篝火，当然这么做只能满足夏思源个人的自嗨并且让更多的玩家不买账而已。

不过，非地图设计上的很多故意恶心人的点就很没必要了。比如比魂 2 开局还慢的喝药，敌我性能如此之高受身居然没有无敌，npc 支线随便就会断掉，即使是老贼也不这么干，你怎么敢的（指指点点

总得来说，很多设计并不能增加玩家的游玩体验，只是魂老害的自嗨，但更多玩家并不接受这样的设计。目前制作组说是会修改这部分问题，但具体会改多少我不好说。这肯定是后续影响评价的重点。

5\. 屁股问题
--------

我懒得讨论这游戏有没有屁股问题，我认为大部分喷这点的压根云都懒得云，少部分稍微云了一点，极少部分玩了还在喷。喷的人就算不喷也根本不会玩也不会买。

但玩了的人里肯定有一小部分人还是觉得有问题并且差评，这也很正常。

### 不同游玩时长下的好评率差异

*   **无明确游玩时长**：整体好评率为 43%，是所有类别中最低的。这可能意味着短暂体验或未明确记录时长的玩家对游戏的满意度相对较低。

*   **10h 以上**：好评率进一步升至 61%，表明当玩家投入更多时间深入体验游戏后，对游戏的好感度持续增加。
*   **20h 以上**：好评率达到 64%，继续保持上升趋势，说明游戏在中长期体验上仍能获得玩家的认可。
*   **30h 以上**：好评率为 69%，再次提升，体现出游戏对于深度玩家具有较强的吸引力和满意度。
*   **40h 以上**：好评率达到 72%，是所有游玩时长类别中最高的，这充分说明长期沉浸在游戏中的玩家对游戏的评价普遍较高。

### 不同语言用户的好评率差异

*   **中文用户**：随着游玩时长的增加，好评率从 39% 逐步上升至 71%，趋势与整体一致，且各阶段好评率均低于对应时长的整体好评率（除 40h 以上外，整体 72%，中文 71%，差距极小）。
*   **外文用户**：好评率整体高于中文用户。无明确时长时，外文用户好评率为 72.43%，远高于中文用户的 39%；3h 以上时，外文用户好评率 83.08%，显著高于中文用户的 50%；10h 以上时，外文 82.91%，中文 58%；20h 以上时，外文 71.96%，中文 63%；30h 以上时，外文 84.76%，中文 67%；40h 以上时，外文 79.28%，中文 71%。不过，外文用户的好评率在不同时长阶段波动相对较大，而中文用户的好评率则呈现更稳定的上升态势。

总体而言，游戏的好评率与游玩时长呈正相关，即玩家游玩时间越长，对游戏的评价越高；同时，外文用户对该游戏的好评率整体高于中文用户。
