在反修例風波和武漢肺炎疫情下，明年應考中學文憑試（DSE）的學生，學習進度大受影響，教育局早前亦表示將研究調整明年的文憑試。《蘋果》獲悉，考試及評核局向學界建議，明年文憑試中文科刪去《岳陽樓記》和《六國論》兩篇範文。  
  
  
![https://images.weserv.nl/?url=https://i.imgur.com/dWpAdJs_d.webp?maxwidth=640&shape=thumb&fidelity=medium](https://images.weserv.nl/?url=https%3A%2F%2Fi.imgur.com%2FdWpAdJs_d.webp%3Fmaxwidth%3D640%26shape%3Dthumb%26fidelity%3Dmedium)  
英國保守黨人權委員會副主席羅哲斯（Benedict Rogers），早兩個月佢先喺facebook出中文練字post，手書《六國論》首4句，呼籲大家停止向中共叩頭。  
[https://m.facebook.com/story.php?story\_fbid=147326086927101&id=102583481401362&sfnsn=mo&extid=9rMugsbbVt9njIkX]( "https://m.facebook.com/story.php?story_fbid=147326086927101&id=102583481401362&sfnsn=mo&extid=9rMugsbbVt9njIkX")  
  
在大多香港學生和香港人，《六國論》是必讀古文。  

```
「以地事秦，猶抱薪救火，薪不盡，火不滅。」
```

  
  

```
以賂秦之地，封天下之謀臣；以事秦之心，禮天下之奇才；並力西嚮，則吾恐秦人食之不得下咽也。
```

  
  
這些都是在主張抵抗，而非綏靖政策，在目前高度敏感的香港，刪除令人感到有政治目的。  
  
香港學者沈旭暉借刪去《六國論》，談論到為何香港比較上多讀古文，反而國內比較少。  
指五四新文化運動後，中國推行新文化運動，而時任港督金文泰本身就熱愛漢學，設立香港首家官立中文學校和支持香港大學增設中文系，也使香港教育比較多有古文，最少基本會文言文。  
除了熱愛漢學外，也有政治目的。因英國不打算買斷新界，那不如借古文來製作文化上的防火牆。  
即使真的如此，金文泰政策也更有利中國古文化的保存。
