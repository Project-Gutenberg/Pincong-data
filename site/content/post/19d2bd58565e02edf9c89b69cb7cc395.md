一、目前显示的 IP 属地信息是否属于法律意义上的个人信息？
------------------------------

根据《民法典》第 1034 条和《个人信息保护法》第 4 条的规定，法律意义上的个人信息应当具有 “**可识别性**”，即要能够单独识别或者与其他信息识别**特定自然人**（后者也被称为间接识别）。[\[1\]]()

目前显示的 **IP 属地信息精确到省级（国内），这样的信息应当不符合上述对个人信息的 “可识别性” 这一基本特征的要求**，因此不属于个人信息。判断是否为个人信息是平台履行 “告知——同意” 的重要前提，我猜测上述用户属地信息保护第一案中**原告小月 “未经过其同意即处理其个人信息” 的观点大概很难得到法院支持**。

二、 是否 “没做亏心事没什么好怕的”？
--------------------

为什么不能将我的所有信息都规定为个人信息？

“没做亏心事没什么好怕的” 是不是一个支持公开 IP 属地的好理由？

支持公开 IP 属地的一点理由是 “没做亏心事没什么好怕的”，在我看来这点理由充满了说教意味，某种程度上也说明了法治意识的树立依然任重道远。“亏心事” 是一个道德的概念，“个人信息权益保护”是法治的范畴，混淆两者的区分固然可以使自己站在虚假的道德制高点，但以自己的权利为代价远非理智。

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-bd857c945e1557ff9238414af3f9d21c_r.jpg%3Fsource%3D1940ef5c)

如上图所示，反对者对这个理由的解构显得过于彻底，虽然表面可能为戏言但却值得认真的讨论。

**受保护的个人信息和不守法律保护的信息之间应当有一条相对清晰的界线**，因为没有这条线只能是两种情况：1）将所有信息都纳入法律意义上的个人信息，这样不仅会使处理和利用个人信息的成本极大，实际上也不利于满足我们生活的需求；2）消解个人信息权益的概念，以 “没做亏心事没什么好怕的” 之类的理由将个人信息权益放弃。

目前来看，立法者把这条界线命名为 “可识别性”。以上图为例，IP 地址、区地址落在了法律所画的对个人信息的保护圈外。反对公开 IP 属地的观点中具有积极意义的一点是对“界线” 的反思：**目前的界线是否合理？**以及 **如何确保这条界线不会随意移动，界线划出的保护圈不会随意缩小？**同样的疑虑还可以推到互联网所讨论的诸多话题：外网和梯子问题？某些作品的审批、删减和下架问题？

三、我对这次网络平台显示 IP 属地问题的观点
-----------------------

之所以最后才亮出观点，是希望能在讨论清楚法律规定等的基础上更理性地给出结论。

**我赞成此次网络平台显示 IP 属地的做法。**理由有以下几点：

1 - 此做法应当能起到净化网络环境的效果。去年开始我不断在思索却至今仍想不出答案的问题是：如何在众说纷纭、反转又反转的网络环境中得知事件的真相？如何超越惯常的站队获得自己理由坚实的观点？如何抑制网暴的出现、减少网暴的伤害？网络平台显示 IP 属地在我看来当然不会有药到病除、解决以上所有问题的疗效，但却是往净化网络方向努力的尝试，这一点值得肯定。

2 - 此做法至少没有逾越现行法关于个人信息保护的底线。具体地看是前述第一部分的分析；抽象地看，**对个人信息权益的界线确定及保护，应当是在价值利用和保护人格两端之间的平衡。（**当然也可以认为现行法关于个人信息保护的规定有误，那么就属于立法论的范畴了，可以再找时间写一写**）。**

法律问题永远不只是法律问题。同其他的举措一样，网络平台显示 IP 属地的做法虽不违法，但可能会出现一些施政者始料未及的负面效果，例如：是否会加剧关于省份问题的拉偏架、地域黑问题，出现 “你是 XX 地方的，怪不得你会支持 XXX，XX 地方的人素质就是不行” 之类的言论？...... 这些伴随着新做法而来的问题也不能因为一句合法而被忽视。

从业务方向理解内容平台显示用户 IP 的策略

微博在今年三月已上线这一策略，同样是识别用户 IP 后展示并且不支持用户个性化修改。从平台安全合规业务的角度理解抖音紧跟这一策略的原因可能两个方面解释；一是上级监管部门的指导意见，二是通过对业务和竞品的综合比对认为这一策略能够有带来正向收益

首先我们要正确理解这里的收益指的是什么。

1.  在内容生产方面，由于增加了这一策略；很多地图炮、境外人士不再肆无忌惮发帖。这一策略可有效减少平台产生的负面内容，减轻机器审核、人工审核的压力，提高审核速度、减少审核成本、在互联网寒冬时节，降本增效。苍蝇再小也是肉，释放的资源可覆盖更多的内容与用户。
2.  这一策略符合监管部门对于各个平台的 “用户合规” 类业务的要求，业务中存在这不合规的情况面对的处罚为罚款、停更、停服等。在各个内容平台明确自身责任和社会义务的情况下，这一策略使风险前置过滤可避免、减轻相关处罚。为平台良性、健康发展提供坚实基础

在这样的大背景下，该策略的上线是充分且必要的，我相信后续有更多的平台会上线该策略。

从长期的业务合规角度来分析，这将成为一个开始。

2021 年国家互联网信息办公室发布了新修订的《互联网用户公众账号信息服务管理规定》，随着互联网推荐算法、内容形式、分发形式的不断发展。越来越多的个人账号开始获得流量倾斜与扶持。具有比肩公众号的浏览量，成为内容消费者获取信息的一部分。所以对于个人账号的合规会不断地清晰。从早期的无需实名制到现在的前端昵称后台实名，成为发展的必然。

对于内容平台来说，这只是一个开始；各个平台将不断落实主体责任，对核验通过后的账号加注专门标识，并根据生产运营者的不同主体性质，在保护个人隐私信息的前提下，公示其内容生产类别、运营主体名称、注册运营地址、统一社会信用代码、联系方式等注册信息，督促账号生产运营者严格自律。
