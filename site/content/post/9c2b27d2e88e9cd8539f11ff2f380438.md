眼镜男叫杨二佳，

几乎有两个排。  
  
「你们跟我来，先去区政府，抓紧时间快下班了。

大哥，刚才那一手怎么来的？

大哥一定要教我，大哥我以后就是你小弟。

"对，我叫 DB， 把手机放电视这里来，

"放心吧，我会帮你找回手机的，

而大金毛已经爬起来围着吉美打转。  
  
在区政府大楼里，

"你还不知道！？

我就知道你们一帮土匪要来区政府。  
  
”我们不是土匪，大哥，

提着冲锋枪上访的？

大哥，不提上冲锋枪的话我们只能去精神病院上访。

就是找清天大老爷告状！

这是小人的诉状，你看。

可是我是个文盲。 厨子说。

直播，直播， 妈的不开直播上访个鸡巴！

你还在吗？  
刚才你说过的明天要给我二百块钱！  
还有我的手机，  
你自己说的。

杨三佳向杨二佳走去给他听自己的手机：  
直播，直播， 妈的不开直播上访个鸡巴！  
  
全部听手机的命令！  
  
二佳已经拿出了手机，  
打开直播模式，  
他一边用手机自拍，

老子是乌黑村的杨二佳，

因为如果老子们集会、示威还有各种和平上访的话，  
老子们们早已经成了严重精神病！  
就算是有8964个老子们也早已经被打死在天安门广场！  
所以老子们来这里上访，  
用这个上访！ 现在，  
想活下去的都给老子老老实实的听 DB 的吩咐！

突突突、突突突...  
一直打光一个弹匣才停下了。  
  
此时除了打碎的天花掉落的声音，  
大厅中只剩下杨二佳更换弹匣发出的咔嚓声。  
杨三佳帮他拿着手机，  
手机里传出 DB 的声音：  
所有人听我的命令！

IT 部的员工跟厨子去设置网络，

再挑出两个女人，

这一幢大楼有六部电梯、三个消防楼梯，

我们抓到了三百名人质，

而所有的人质都在乌黑村土匪凶恶的注视下听着自己的手机，

...... 爸爸，求求你千万不要让人断网、断信号，  
我们一定会和劫匪斗智斗勇，  
求求你千万保住网络和信号，  
不然我坚持不下去......一个女人在打电话...  
  
......妈妈，劫匪只要美金，  
他们说是一千美金放一个人，  
为什么我这么便宜？！...  
不，没有，他们没有强奸任何人......一个男人在打电话...  
  
在吉美的手机上，  
DB 继续给厨子讲解着：  
这一座大楼东面一百米有一座五星级酒店，

2020年 7月29日，  
Google， Facebook， Amazon， Apple，

不过美国的反托拉斯法大部分是在一个世纪前订定，能否有效规范网际网路业者，恐怕大有问题。29日听证会进行时，川普上推特警告，如果国会无法为科技业巨子带来公平性，他会对业者祭出「行政命令」杀手锏。

DB 还没有说完，

youtube, facebook, twitter, telegram,

其中有杨三佳的征兵广告：

还有几十把 M16, M4, M203 放在最显眼的位置。  
他问： DB， 又是你的谋划？  
  
DB： 当然，以后叫我大哥大！  
  
我忽然想起来，在我变成狗之前，  
女王要我来寻找让这个国家学会「上跪下」的方法，  
可是现在却在这个地方打得乱七八糟没头绪了。  
  
DB：有枪就有上跪下，  
不信我们拿上枪上楼，  
市长什么的都要给你跪。  
  
我是文盲可是我并不傻呀，  
陛下好像不是这个意思。  
  
DB: 你以为我不懂中国人下跪是什么意思吗？  
膝盖和额头都要碰在地板上！  
  
先去吃早饭吧，对了，  
DB 你知道一顿美味的早餐是什么感觉吗？  
  
DB： 你给我把手机充满电，还有所有的电脑都要开机！  
  
说着 DB 已经把数千道指令发去了乌黑村土匪和人质们的手机上。  
  
小娟收到信息就把手枪藏在公文包里起身向五星酒店走来。  
她的手机上又弹出二大爷的两篇文章：  
...大清挨揍，百姓都干嘛去了？  
八国联军起兵始末...  
然后墙内有爱国学者大量的反驳文章，  
因为反驳二大爷最正义。  
  
从秦朝到中共红朝，  
中国人从来难以向上证伪。  
而证伪能力却是科学探索的基础，  
因此中国没有发展科学的机会，  
这是历史事实。  
  
厨子不知道 DB 同时在处理无数的线索，  
在餐厅门口他对 DB 说：  
当你把金字塔放在证伪的海水中时，  
我已经知道不需要这样打打杀杀了。  
因为一道接着一道的潮水早晚泡烂金字塔。  
  
DB： 那么你也不用去找白洁了吗？  
  
DB： 其实，我同时在计算中共崩溃的时间次序。 你知道太平天国的时候， 1853年打到南京。  
清朝设立两大军营围杀起义军，  
江北大营、江南大营。  
  
在江北大营，军官盘剥士兵军响，  
士兵出营沿路设卡剥削路人、商人，祸害地方。  
江南大营也不会好多少，  
军营里都藏污纳垢。  
  
当太平军打破江南大营时，  
逃出来的兵士竟然有一半是女人男装，  
江南江北就根本是两座淫秽的军营，  
你说这里发生多少黑暗的故事？  
而今天的腐败如同历史重演。  
  
虽然说太平天国自身装神弄鬼也有先天不足，  
然而我要说的是：  
太平天国被屠灭之后慈禧太后掌权，  
清朝续命60年。  
  
现在！我的问题就是：现在资讯传播速度比清朝快不只一百倍，  
会不会还有 60/100 年就有中共红朝覆灭。  
  
厨子笑了。  
  
DB：我说的就是中共再有半年多就要灭亡了，你不激动吗？  
  
厨子无语。  
  
DB： 你看，我一出现就有天地巨变。你为什么不激动？  
  
厨子： 你这样计算的话，  
是说我们先要打一场太平天国吗？  
也好， 以后你叫我： 天王！  
  
小娟已经走到餐厅门口，笑问道：  
厨子哥，你为什么挡住餐厅门？  
  
这时候厨子手中吉美的手机发出吉美的老师的责问：  
吉美同学，你为什么没有在同学圈表示决心要打倒美国大流氓？  
  
小娟听到了就问厨子「吉美是你的女儿吗？  
每个学生都要坚定决心打倒美国。  
你知不知道美国关闭中国在休斯顿的领馆之后，  
中国关闭了美国在成都的领馆，  
中国舆论群情义愤，  
人民表示包子将领导中国战胜美国...  
学生们都在狠批美帝，仇恨美国...」  
  
厨子忙说：我不想吉美变成这个样子。  
  
小娟说：那么你的父女关系就会有问题了。  
  
DB 急忙插入： 就是说吉美会来教育你，  
你需要仇恨美国，老师们会派她来完成任务，  
教育你。 然后，你当然会变成不中计的顽固派，  
但是吉美就很惨了，她有一个舔美的无耻父亲，  
一个没骨头的男人，人渣， 一个文盲，  
哈哈， 就是你啊... DB 开心的说， 哈哈，哈哈  
  
就在我家里做愚民的工作？  
就要把吉美变成我的敌人？  
我不同意。 厨子狠狠的说。  
  
小娟有趣的看着厨子，  
这时候厨子看到吉美奶奶的号码拨入，  
然后是吉美的声音：  
大叔，你还拿着我的手机吧，  
请帮我在朋友圈发言，  
美国帝国主义夹着尾巴逃跑了，  
我最爱包子，我最开心关闭他娘的美国大使馆，  
全部给我关闭......  
大叔不要忘记了，  
我奶奶会检查我的朋友圈的。  
我们祖国取得了伟大胜利，  
你不要忘记了，奶奶会检查的！  
  
厨子抓着手机大叫着走出餐厅：  
吉美变成小粉红了？！ DB！  
  
小娟跟来安慰他：厨子，  
孩子需要有正确的学习态度，  
你不要太激动，  
不然你的女儿可能没有机会考上大学。  
  
  
  
  
八  
\--  
  
  
"你不感觉这样过于魔幻了吗？  
我们抢下来一座政府大楼，一座五星酒店，一座水疗城，  
还有五百名人质，和两大车武器弹药，  
我们还给全世界发出了征兵广告，  
而你现在说你就要走了！  
你还说你要先回去归还一个手机？！  
给一个叫什么吉美的中二女学生？"  
\-- 杨二佳不可置信的瞪着厨子大喊。  
  
DB 的声音也清晰的插入：  
"并且你忘记了吗？  
吉美并不认识你，  
你不过是偷了她的手机的一个人。"  
  
厨子楞了一下：  
"是啊，吉美还不认识我。"  
说着一伸手就拨出杨二佳的手枪，  
一支 Glock 21，  
他抬头对着酒店大堂顶上的吊灯射击，  
大堂里一声接着一声枪响，  
吊灯的繁复花枝一根接着一根的碎落到地板上，  
很快打空了一个弹匣，  
他换手抓出了杨三佳的 1911，  
又是一支 .45 的大口径手枪，  
这是一把钢材更多的老枪，  
厨子继续用子弹修剪吊灯的花枝，  
直到弹匣再次打空，  
金属弹壳落在大理石地面上的清脆尾音被吊灯花枝摔碎的碎玻璃声音掩盖，  
他野蛮的问：看我像不像土匪？  
  
说完就溜走了。  
  
"你去哪里？ 你回来！... 他溜走了？！"  
杨三佳向四周大喊了几声，  
他不知道是不是应该去追上厨子。  
  
奇怪的是大家都见怪不怪，没人感觉意外。  
一群吃过早餐的土匪过来看看也就走了，  
他们都拿着手机。  
  
DB 非常重要并且 DB 无所不在。  
DB 在每一个人的手机里，  
所有人都信任 DB 但是没有人知道 DB 是什么。  
  
在满地碎片的酒店大堂里杨三佳哇哇大叫了一会儿就看着杨二佳，  
两人大眼瞪着小眼， 然后他们的手机开始说话：  
「啊， 三佳同学，  
厨子会回来的，你要相信我。  
现在，你们有吃有喝有三座豪华建筑，  
不忘初心实现梦想的燃情岁月就是现在啊，  
安心做几天奢侈的土匪吧。」  
  
可是我们能吃到今天的午餐吗？  
  
杨二佳也问自己的手机：他能去哪里？  
现在外面至少几千人大部队围紧了吧？  
  
「这些都不重要，  
让我继续给你们讲金字塔的原理，  
如果你们明白金字塔上面就是地狱的入口，  
你们就应该知道自己已经成为地狱战警，  
相比之下午餐什么的真是不值得担忧。」  
  
DB, 你可以说点有用的吗？  
  
「金字塔里说什么有用呢？  
做人的道理搭配的各种鸡汤在金字塔上都是有毒的，  
唯一有用的就是造反。」  
  
杨二佳耐心地把一颗颗子弹压回厨子打空的弹匣中，  
他敲敲弹匣说：所以我们乌黑村的杨家军就是唯一有用的中国人？  
  
「严格的说不算是唯一，  
金字塔是无法被子弹打碎的，  
只有一波又一波的带着证伪之力的潮水可以泡烂它...,  
算了，不废话证伪金字塔的事情，  
总之，我会让你们有吃有喝做英雄...」  
  
现在只有相信你们了，对不对？  
  
「对，这是最正确的事情。」  
  
  
在吉美家的小店里。  
  
大金毛犬开心的扑在厨子身边，  
厨子一边吃小笼包子一边和吉美说话：  
包子真好吃...  
  
「大叔，你是怎样捡到我的手机的？」吉美忍不住又问一边。  
  
「就是路边捡到的。  
可是，你是怎样捡到我的大金毛的？」  
  
大金毛也转头认真的看着吉美。  
  
「啊，」厨子突然又说：  
「没关系，没关系，  
其实我总不在家，  
我的狗养成了习惯自己乱跑，  
我很想谢谢你呢。  
还有包子真好吃，  
以后我每天来吃好吗？...」  
  
酒店里，DB 运筹帷幄。  
  
「二佳，厨子根本是个没用的家伙，  
以后你要习惯啦，  
不要以为自己做了多大的事情，  
造反这样的事情古人比你厉害多了。」  
  
可是我根本不知道你是哪里来的，厨子我还看得见，  
而你就突然把我们的手机全部变成精了，  
当然结果是很好，但是你能让我知道你是哪里来的吗？  
  
「我是 AI 你如何看到我？  
你要习惯于人类科技的进步，  
就算你不信神，  
也要知道我值得你的膜拜！  
下面给我听好了，  
金字塔里重力是向上的，  
畜生轮奸了学者，  
学者阉割了同胞的证伪能力，  
在这里善良就是邪恶，  
智慧就是阴谋，  
最诚实的就是谎言，  
这里的成功者都该下地狱，  
而金字塔也正是把所谓的成功者一步步拉上地狱的台阶，  
一个也不能少。」  
  
地狱是爬上去的？ 我一直以为就摔下去的。  
  
「对，地狱是爬上去的，  
你又知道了一个秘密，我对你寄予厚望。」  
  
可是还有好人，  
昨夜在水疗会所我搂着个小姐，  
她的奶子又白又嫩，  
一晚上都在劝我热爱祖国改邪归正，  
真的比我妈还好。  
  
杨三佳凑过来说：  
二哥，这会儿就想多活几天了。  
  
杨二佳冷冷地看了他一会，  
又对 DB 说： 我们拉几十个兄弟来造反也不容易，  
DB 你说我们还有活路吗？  
  
「有。 逃离金字塔我最专业。」  
  
老大，你最喜欢讲金字塔，我只想活下去。  
  
「你给我听好了，我不想重复，  
你端着枪冲进政府大楼的时候你想过要活下去的事情吗？  
你给我听着，金字塔是人类建造的最差劲的建筑，  
尤其是以人类的血肉做建筑材料的时候。  
人类文明的足迹从两千年前起就是跟随着耶酥信徒的足迹，  
而在中国拆卸十字架的人却可以升官，  
你知道去香港做官的骆惠宁吗？」  
  
老大， 我不信神。  
  
「耶酥是个历史人物。  
现在， 我有其它任务， 再见。」
