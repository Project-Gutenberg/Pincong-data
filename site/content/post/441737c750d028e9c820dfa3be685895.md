过去半个月中国疫情越来越严重，政府手忙脚乱，再加上春节假期，一时间舆论管控居然松了不少，微博豆瓣等地方出现了很多沉默已久的声音，不过根据最近的小道消息来看，中央又要开始严管舆论了。  
  
可惜瘟疫和其他事情不一样，上海有人抗议，北京人可以不管不问，毕竟影响不到自己，让中央直接把上海维稳就行了。瘟疫就没那么听党的话了，领导来了也会亲自感染，所以为了生命安全，获取真实信息就尤为重要。  
  
这段日子我从鹅组那里看到不少新闻，也搬运到品葱一些，也算受益者，所以今天就手把手教鹅组姐妹注册品葱和 Telegram ，算是回报。  
  
1\. 注册品葱（[https://pincong.rocks](https://pincong.rocks "https://pincong.rocks")）  
  
首先，访问品葱一定要注意安全**，至少要用 VPN + Tor Browser 来访问，有条件的请使用虚拟机**。（[如何使用 Tor](https://pincong.rocks/article/10677 "https://pincong.rocks/article/10677")）你可能会问，品葱网站本身安全吗？会保存用户信息吗？我以前写过一篇《写给香港手足的品葱简介》（[https://pincong.rocks/article/9958](https://pincong.rocks/article/9958 "https://pincong.rocks/article/9958")），摘录其中的安全说明：  
  

*   注册完全匿名。不需要邮箱，也不需要手机号。
*   服务器不记录 IP 地址。
*   发帖和回复都可以设置延时发布。别人通过发帖也无法判断分析你的行为。比如我就常用延时，如果警察看我发帖，会有一种我二十四小时都在线的感觉，不知道我的位置。
*   混淆时间线。有时候回复的日期也是打乱的。
*   用户随时提醒你不要泄漏身份信息。这里的用户都是身经百战，有的还因为不小心被警察找上。
*   注意用户隐私，默认搜索也是 DuckDuckGo

  
  
做好安全措施之后，就可以开始注册品葱了。注册不需要提供邮箱，只需要用户名，非常简单。但还是要注意：注册时**千万不要使用任何和个人有关的信息，比如名字拼音，比如曾用网名**。我知道用户名不太容易想，怎么办呢？这里手把手教大家如何获得一个安全的用户名。  
  
各位去豆瓣鹅组，豆瓣七组，或者豆瓣其它小粉红聚集地去搜索“带节奏”，如图：  
  
![https://images.weserv.nl/?url=https://i.imgur.com/giaTiDO.png](https://images.weserv.nl/?url=https%3A%2F%2Fi.imgur.com%2FgiaTiDO.png)  
  
然后就能发现很多网评员或者粉红发的帖子了，进去之后浏览一下，看有没有像样的 ID，比如这个（该用户发现了本文，已经主动注销了豆瓣账号，哭哭）：  
  
![https://images.weserv.nl/?url=https://i.imgur.com/6eXlO5i.png](https://images.weserv.nl/?url=https%3A%2F%2Fi.imgur.com%2F6eXlO5i.png)  
  
那我就用它的 ID 来注册品葱好了。为了防止自己把 ID 忘掉，可以先把此人拉黑，以后永远都可以在黑名单里找到。  
  
![https://images.weserv.nl/?url=https://i.imgur.com/ASHZdaK.png](https://images.weserv.nl/?url=https%3A%2F%2Fi.imgur.com%2FASHZdaK.png)  
  
注册成功之后呢，如果你觉得需要一个头像，也可以翻翻此人的相册，找一张照片保存下来，然后上传作为自己的头像。  
  
![https://images.weserv.nl/?url=https://i.imgur.com/FeXrVUm.png](https://images.weserv.nl/?url=https%3A%2F%2Fi.imgur.com%2FFeXrVUm.png)  
  
这个照片就不需要注意安全了，最好是这个人的私人照片，照片里面最好带有个人信息。如果你不小心在品葱发表了反动言论，被“经验老道”的网警发现，就当你请这个粉红喝茶了，哈哈。  
  
  
2\. 注册 Telegram  
  
**任何时候，一定不要暴露自己的手机号码，务必在隐私设置中，设置手机号码仅自己可见。**  
  
很多人用国内手机号（+86）注册 Telegram，虽然 Telegram 不会泄漏你的号码，但**国内运营商可以获取它发给你的验证码**，所以还是不安全。所以最好用虚拟号码注册。免费的虚拟号码不难找，老牌的有 https://textfree.us/ ，今天介绍一下我在用的另一个服务：MySudo https://mysudo.com/  
  
MySudo 是专门为了保护隐私设计的，帮你创建匿名身份。下载之后不需要任何个人信息，打开就可以获得一个免费的虚拟号码还有一个免费的匿名邮箱。大部分网站注册时都会需要手机号或邮箱，所以你就可以用 MySudo 为中心，建立一个完全虚拟的身份。免费版有很多限制，不过我觉得也够用了。  
  
MySudo 目前只支持 iOS。如果在中国区的 Apple Store 下不到，可以考虑把 Apple ID 转到美区，转区可以用 iCloud 3.0，不需要信用卡。  
  
一旦获得虚拟号码，注册 Telegram 就很容易了，具体教程大家去网上搜吧，墙内也有很多。  
  
  
多说一句，很多在墙内生活过久的人，刚出来面对各种信息冲击会特别不适应，可能会被各种极端言论吓到，然后想缩回去。没错，墙外也有很多虚假信息，也有各种极端言论，但是我们恪守言论自由这条底线，就会想办法提高自己和身边的人的分辨能力，努力做好 fact check，想办法怎么改善教育，然后整个社会一起进化，而不是盖一道墙把自己关起来。  
  
经过这场瘟疫，相信稍微有点智商的都能明白，哪怕言论自由会纵容很多“有害信息”，也比独裁和言论管制好得多。好好想一想，如果中国这几年有 journalism ，这场瘟疫会不会不一样？
