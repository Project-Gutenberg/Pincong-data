金浩阳毛巾，永亮毛巾等用便宜的纱线加工，根本达不到国家标准。那些厂家人士更是说，用来制作毛巾的纱线都是用低劣的再生棉制成的。

业内所说的再生棉，指的是一些纺织厂的各种下脚料、碎布片、线头等经过再梳理、再开花，就成了再生棉。而再生棉的原料有旧衣服、毛衣、外套、秋衣秋裤，甚至穿过的内衣、袜子等。

**为什么这样干？还不就是利益熏心**，“正规的棉花一吨 16000 多元，14 支纱线的下脚料多便宜，但脱毛率没一个达标的。” 据央视援引一位纺织品公司总经理的话称，这种达不到检测标准的毛巾，他们一年的销售额大概能卖到 5000 万元。

现在，在淘宝、京东、拼多多等电商平台上搜索 “永亮毛巾”、“金浩阳毛巾” 等关键词，已搜索不到相关产品。

覆盖了很多 App 以后他们发现，他们收集的数据非常非常多，可以看出整个移动互联网的发展趋势了，于是开始做数据变现。

怎么变现呢？

第一步，先加密存储，所有的数据采集以后都是加密的，给你看仪表盘，但是不给看源数据，如果要看源数据，另外收费。

第二步，由于他们知道很多 App 的安装情况，使用情况，同一个手机上可能有多款 App 有他们的 SDK，按照同一套设备指纹算法，他们就知道装了 A App 的人，还装了 B App，这就可以做新客营销了，如果大多数人都是既装了 A，又装了 B，这时候突然有几个人，只装了 A，没有安装 B，那就可以推荐这几个人装 B（此装 B 非彼装 B……）。当然，客户的设备信息全部加密存储，无法定位到一个具体的人，只能知道某个设备，相对来说一定程度上保护了隐私。

第三步，知道很多 App 的安装情况，也可以获取到定位，于是我们可以绘制出热力图。热力图能干嘛？热的地方说明人多，可以帮助线下店铺选址

**8.5 个人信息主体注销账户**  
对个人信息控制者的要求包括：  
a) 通过注册账户提供产品或服务的个人信息控制者，应向个人信息主体提供注销账户的方法，且方法简便易操作；

d) 注销过程不应设置不合理的条件或提出额外要求增加个人信息主体义务，如注销单个账户视同注销多个产品或服务，要求个人信息主体填写精确的历史操作记录作为注销的必要条件等；

注 2：产品或服务没有独立的账户体系的，可采取对该产品或服务账号以外其他个人信息进行删除，并切断账户体系与产品或服务的关联等措施实现注销。

**曝光毛巾黑暗面：**

**旧袜子旧内衣竟是生产原料**

河北省高阳县是中国最大的毛巾生产基地，年产毛巾 50 亿条，约占全国的三分之一。然而记者走访了高阳，却了解到虽然标注着执行国家标准，但有的毛巾实际上却无法保证检测合格。

那么，这种纱线用的是什么原料、如何生产出来的呢？

在高阳县辛留佐村，记者找到了一家专门生产毛巾纱的企业，记者看到，工厂生产车间有一个配料池，池子里是用来纺毛巾纱的原料，有的上面还沾染着绿色的污渍，有刺鼻的味道，里面的杂质清晰可见。负责人冯经理告诉记者，这些原料大部分是再生棉。业内所说的再生棉，指的是一些纺织厂的各种下脚料、碎布片、线头等经过再梳理、再开花，就成了再生棉或者叫回收棉，这种棉花比新的棉花便宜近一倍。高阳及周边多家为毛巾厂生产纱线的企业，每家产量少则一百吨，多则数百吨，那么，这些再生棉，又是如何加工而成的呢？

在河北邯郸南井寨村，这里生产再生棉的原料让人震惊。厂子里的旧衣服堆积如山，有毛衣、外套、秋衣秋裤，甚至穿过的内衣、袜子等。**在没有经过任何消毒程序后，经过机器绞碎、几道开花程序，旧衣服重新变成了棉花**，同时，产品包装上，依然没有任何警示标志。

**曝光手机里的窃贼插件：你的短信可被全部传走，包括网络交易验证码**

SDK，是在手机软件中，提供某种功能或服务的插件，2019 年 11 月，上海市消费者权益保护委员会委托第三方公司对一些手机软件中的 SDK 插件进行了专门的测试，却发现一些 SDK 暗藏玄机。

技术人员检测了 50 多款手机软件，这些软件中分别含有上海氪信信息技术有限公司和北京招彩旺旺信息技术有限公司两家公司的 SDK 插件，**这两个插件，都存在在用户不知情的情况下，偷偷窃取用户隐私的嫌疑**，涉及国美易卡、遥控器、最强手电、全能遥控器、91 极速购、天天回收、闪到、萝卜商城、紫金普惠等 50 多款手机软件。

![](https://images.weserv.nl/?url=https%3A//pic2.zhimg.com/v2-5606e26607ea36a92cbecb69244d28dc_r.jpg%3Fsource%3D1940ef5c)

检测人员：“**它会读取这部设备的 IMEI、IMSI、运营商信息、电话号码、短信记录 、通讯录、应用安装列表和传感器信息。**”

这些 APP 里的 SDK 来读取用户的隐私信息只是第一步，读取完成后，还会悄悄地将数据传送到指定的服务器存储起来。北京招彩旺旺信息技术有限公司的 SDK，甚至涉嫌通过菜谱、家长帮、动态壁纸等多款软件，窃取用户更加隐私的信息。

检测人员：“**会未经用户同意，收集用户的联系人、短信、位置、设备信息等等。尤其短信，短信内容被全部传走，这个是很严重的。**”

检测人员介绍，因为 SDK 能够收集用户的短信，以及应用安装信息，一旦用户有网络交易的验证码被获取，极有可能造成严重的经济损失。

![](https://images.weserv.nl/?url=https%3A//pic3.zhimg.com/v2-90921c05525322dda894471571991d2a_r.jpg%3Fsource%3D1940ef5c)

此外，虽然 SDK 只是一个看似普通的插件，但是因为它对所有的手机 APP 具有通用性，很多手机软件可能都嵌入了同一个 SDK，因此一旦某个 SDK 窃取用户个人隐私，将会涉及众多手机软件。在此次检测当中除了内嵌 SDK 插件以外，工作人员还发现一些知名手机 APP 也有收集用户隐私的现象，涉及酷音铃声、手机铃声、铃声大全等多款软件。

作为一名安全从业人员，在我看来，其实 315 晚会当天最值得关注的信息，就是没有出现在节目预告中的个人隐私信息被窃取问题。

上海氪信信息技术和北京招彩旺旺信息技术这两家公司，通过在 50 多款手机 APP 中植入 SDK 插件（一种软件包），在用户不知情的情况下，偷偷窃取用户的隐私，包括设备的 IMEI、IMSI、运营商信息、电话号码、短信记录 、通讯录、应用安装列表和传感器信息。

更可恶的是，这些隐私信息被读取完成后，这两家公司还会悄悄地将数据传送到指定的服务器存储起来，一旦用户有网络交易的验证码被获取，极有可能造成严重的经济损失。

除了获取短信验证码等隐私信息外，大家可能还不知道，如果这些 SDK 插件能读取手机用户的传感器信息，就意味着它可能会 “偷听” 大家。

这里给大家科普一下，手机里的加速度传感器主要用于运动计步。它的工作原理和麦克风一样，都是记录震动。每当手机扬声器发出的声音，这些震动都会被加速度传感器记录下来。只要把这些细微的震动变化进行还原，就可以识别破解出扬声器里的谈话内容。

更可怕的是，手机 App 对加速度传感器的调用，并不是高权限，可以在不询问的情况下悄悄开启。在这种情况下，不论是苹果，还是安卓手机，都难逃被偷听的命运。

不久前，浙江大学发布了一项关于手机偷听的研究成果，智能手机 App 可以在你毫不知情的情况下，利用内置加速度传感器，实现对用户语音偷听，且识别正确率高达 90%。

### 海参水「深」

为了清除海参生长水域的其他生物（鱼、虾），部分不法海参养殖者将大约每亩 2 公斤的敌敌畏投入了养殖场，把除海参之外的生物都药死。之所以大胆地往水里扔敌敌畏，也是因为海参有很强的抗药性，很难药死、药伤。而在养殖后，这些敌敌畏浓度极高的水还会被「放走」，重新流入河海。

### 毛巾的正反面

看完下面这条调查，你可能不敢用毛巾了。

在河北高阳，全国每三条毛巾就有一条产自这里。但央视经过调查发现，这个全国最大的毛巾生产地，却在用着来源不明的下脚料、碎布片、废线头和旧衣服来生产毛巾，基本都不符合国家检测标准。

更加触目惊心的是，这是整个毛巾上下游产业链的共谋。

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-b98621875b6c2fb49ba35d87facb57f6_r.jpg%3Fsource%3D1940ef5c)

据悉毛巾的质量由纱线决定，然而当地不少工厂却在用着不合格的再生棉来生产纱线。所谓再生棉，就是用是由废弃的棉花、工业下脚料和纺织企业的布头和纱线头回收再利用制成的棉絮。

由于经过多次循环利用，可以看到这些再生棉大都充满杂质，甚至发霉。在一些生产工厂， 包括内衣在内的一大堆旧衣服， 没有经过消毒直接搅碎变成棉花 。这些再生棉不会标注原料、成分、厂名厂址，是彻头彻尾的三无产品。

根据国家相关标准，再生棉这种再加工纤维，是禁止用于毛巾这类直接接触皮肤的产品及生活用絮用纤维制品的。这些工厂之所以敢无视国家标准，说到底还是因为利益。再生棉的成本比合格的棉花便宜一倍，由此带来的便是巨额利润。

![](https://images.weserv.nl/?url=https%3A//pic3.zhimg.com/v2-f828d35142abe6a2f9a793dc57998f7a_r.jpg%3Fsource%3D1940ef5c)

据部分原材料和纺织工厂透露，这些用再生棉制作的毛巾，能给他们带来五六百万到五千多万的收入。

你现在的用的毛巾，说不定曾经就是某人的内衣裤。

### 精装修成了「惊」装修

说完了车子，说大家关心的房子。

近年关于「精装修」的投诉层出不穷，很多可能就发生在你身边。

消费者们本以为花高价买了知名企业的精装修房，就能省时又省力，结果买完后很多成了大麻烦。

万科每平米几万的精装修房，就在 2018 年被爆出经过房间的蓄水测试后，201 户里面有 146 户出现了漏水的情况，墙壁被积水渗透、排污管飙粪水、卫生间成了水帘洞。最让人担忧的还有安全问题，因为灯泡里面甚至都会水流不止。

![](https://images.weserv.nl/?url=https%3A//pic3.zhimg.com/v2-e42cdf24e9444f3f85743fbee28f6853_r.jpg%3Fsource%3D1940ef5c)

不止漏水问题，很多业主还反映地暖开几天后，地板变黑；煤气管道没有接稳，造成煤气泄漏等等，甚至到了收房前，地板都是拆卸的，厨房门没安好，地坪也都是潮湿的。

这些问题少则几十个，多能达到几百个，但客户经理却一再拖延和敷衍回应。

更令人不解的是，在杭州春秋华庭小区等地，同样的问题一堆的情况下，业主还会被开发商变相催促强制收房，表示收房后再处理房屋问题，所有都得同步进行。

![](https://images.weserv.nl/?url=https%3A//pic2.zhimg.com/v2-056cd88f3cf5e4e931c427fb66272d12_r.jpg%3Fsource%3D1940ef5c)

「开发商傲慢的态度让人愤怒失望到无助绝望，强制交房更是让人心寒」，业主们说道。

事实上，这还是因为精装修产生质量问题之后，对业主的补偿并没完善的制度规范，要解决问题不止需要政府相关部门加强管理，同时也需要开发商严守自己的行业规范，诚实守信，才能赢得市场口碑和消费者认可。

6、**检验空调机位是否均有设置空调排水管**，检验马桶蓄水速度及排水速度，检验是否存在水电相邻布置有安全隐患。　　

**“电” 路工程检验**

1、入户的总用电容量一般都有限额（一般家用平层不低于 10 平方），**“照明、插座、厨房、卫生间、空调专线分路” 是最基本的要求，**这些都应该能在配电箱的线路布置中体现出来。

2、**检验拉闸断电实验，**首先拉下总闸看户内是否完全断电，再分别关闭分闸，看是否完全控制各分支线路。

3、**检验各屋灯头、插座是否有电，开关是否正常、电话插口、宽带插口、电视天线插口、急救插口是否位置准确通畅，必要的可以协同开发商一起检验。**

1、**天、地、墙的施工质量，会直接从房子的各个表面体现出来，对房子的美观和功能有很大的影响。**对 “天、地、墙” 的验收会有许多细节问题需要注意。首先检测地面是否平坦，专业人员是用水平仪，大家可以用一个乒乓球来代替，把乒乓球放在地面上，如果乒乓球乱滚说明地面倾斜比较严重。

2、**注意看墙体转角线是否笔直，墙体是否平整，是否存在裂缝**，如果发现一定要弄清楚是否系房屋结构性裂缝，最好叫开发商来检查处理。

3、**要检查墙面地面是否存在空鼓，**只要用空鼓锤轻击，就能听出下面有没有空鼓了，一般墙砖要求较高，影响安全的不允许空鼓，地砖主要行人通道不允许空鼓，总体而言国家规范中关于空鼓要求不能大于 5%。

6、**看看地板是否起翘，听听地板是否空心，**留意地板有无明显的缝隙和不平整，地板与踢脚板接合部是否密合，在每个房间里慢慢地走一遍，仔细听听有无响声。

7、漆面查验可以在 1 米外环视，不允许有色差、污渍、泛碱、流坠、疙瘩、沙眼等。
