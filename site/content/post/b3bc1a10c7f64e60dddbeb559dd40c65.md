但是为什么现在电商税闹得沸沸扬扬呢？

因为这里面涉及到了一个核心问题，就是追溯三年，没有[成本票](https://zhida.zhihu.com/search?content_id=756909397&content_type=Answer&match_order=1&q=%E6%88%90%E6%9C%AC%E7%A5%A8&zhida_source=entity)的电商，按 13％增值税硬交，很多商家就得暴毙了。然而这个做法实际上已经是对商品重复征税了。

本来一个商品流通一次，有成本票，只是对增值部分缴税，也就是一块钱买进来两块钱卖出去，只需要交赚了这一块钱的 13％，也就是一毛三分钱。但是，如果你没有成本票，那你就得交两块钱的 13％，两毛六。这两毛六事实上已经包含了之前所有流通环节的增值税部分。而这么多环节全部不交税几乎是不可能的。

中国电商的产品为什么那么便宜，不仅是流通环节少了这一个原因，电商在过去这里面里税务机关选择性忽视也有重大关系。

也就完成了中国社会的一个普遍现象——严格立法，选择执法，普遍违法。

### 知乎用户 OwlLite​ 发表
    
个人认为 “[电商税](https://zhida.zhihu.com/search?content_id=757292992&content_type=Answer&match_order=1&q=%E7%94%B5%E5%95%86%E7%A8%8E&zhida_source=entity)” 不应当单独的拿出来看，而是放在当前的经济背景之下看更合适。

具体地，当前中国 GOV 的负债率已经攀升到相当长历史中的极值位置，并且还在提升，特别是 LGOV 的杠杆率已经大部分超过 500%. 在这种情况下，十五五规划中明确了进行 “梭哈式” 科创投资，寄希望于此挽回经济上行趋势。但稍微有一点点参与这其中的投资和研发经验就可以知道，这种大规模的 “梭哈式” 科创投资本质上是 政府和相关势力圈钱的一种相对新的包装方式，并非为了产生经济价值或者产业安全。也就是说，在这种科创方面 “大炼钢铁” 是 以 “正当性” 和“合理性”包装之下，还是经久不衰的老一套权利寻租行为。

在当前的 [PPI](https://zhida.zhihu.com/search?content_id=757292992&content_type=Answer&match_order=1&q=PPI&zhida_source=entity) 长期收缩，房价下跌、租金下滑、就业情况恶化的情况下，居民部门开始了大力度的主动去杠杆过程（参考 10 月央行数据）。但是，刚过去的 10 月，个人所得税也同比增加超过 27%，远超过工资收入的同比增速，这明确地说明，电商税（增值税的电商场景落地）并非是当前环境下的个例，而是地方财政压力增加情况下 “不得不做出” 的操作。这种实质性、普遍性的加税，会加剧居民部门的 Balance Sheet Recession 深度和持续长度，压低居民部门的可支配收入增长预期，从而压制就业和消费，削减个人所得税和电商税（增值税）的税基。

所以，当前情况虽然很多人说是 Balance Sheet Recession，但实际上会更接近于 “**[财政紧缩](https://zhida.zhihu.com/search?content_id=757292992&content_type=Answer&match_order=1&q=%E8%B4%A2%E6%94%BF%E7%B4%A7%E7%BC%A9&zhida_source=entity)下的[通缩螺旋](https://zhida.zhihu.com/search?content_id=757292992&content_type=Answer&match_order=1&q=%E9%80%9A%E7%BC%A9%E8%9E%BA%E6%97%8B&zhida_source=entity)**”。这种组合在经济学上极其危险，在现代社会例子也不多，因为它意味着在私人部门去杠杆的同时，公共部门不仅没有有效地提供需求（因为投资效率低），反而通过实质加税进一步抽取了市场的流动性——这意味着当前环境下最恶劣和最愚蠢的政策组合。

可以预期的是，就业和税基会因为当前实质加税的操作而被削弱，虽然当前收到的税增加，但是很快会陷入下降，GOV 部门可能会不得不更早地推出房产税以填补 LGOV 的亏空。也就是说，房产税是有可能在房产价格下滑企稳之前推出的。

因此，“电商税”虽然惊人，但不过是 财政顺周期性在向上负责机制下的自然结果。当税基收缩到极限，一些变化不得不做出，才意味着新的周期可能的开始。而 “电商税” 和在这个下行周期下失去自己青春和机遇的一代（甚至数代）人，不过是这场悲剧容易被忽略的注脚罢了。至于相应的房地产超出想象的持续暴跌，在最后可能只是众多可怕后果中不起眼的那一个。

我觉得很合理，实体行业要交税，为什么电商行业就能例外？

第一、有效抑制从生产端就开始的偷税漏税现象
---------------------

电商税一旦正式实施，首先会倒逼电商经营者在进货时主动索取发票。这样一来，上游生产商的偷税行为就会大大减少。举个例子：

**假设某商品进货价 100 元，售价 120 元，利润 20 元。**

**如果进货时没有发票，按 13% 的[增值税](https://zhida.zhihu.com/search?content_id=757354493&content_type=Answer&match_order=1&q=%E5%A2%9E%E5%80%BC%E7%A8%8E&zhida_source=entity)率，需缴纳 15.6 元税款，电商实际利润只剩 4.4 元。**

**而如果进货时要求开票，供应商可能会把价格提高到 113 元，但电商仍卖 120 元，利润为 7 元，此时只需缴纳 0.91 元增值税，实际利润反而有 6.09 元。**

看出来了吗？索要发票虽然提高了进货成本，但在同样售价下，电商的税负降低，利润反而更高。

这会产生强大的 **“传导效应”**。

数以百万计的电商卖家，为了自身的生存利润，会形成一股强大的合力，倒逼他们的供应商、生产商必须开具合法发票。

这将从源头上大幅压缩生产环节偷税漏税的空间，让国家税收的变得更加畅通。

我们平时总批评某些企业偷税漏税，电商税正是解决这一问题的制度设计。

第二，电商税还能有效遏制假货和劣质商品流通
---------------------

以盗版书为例：一旦所有电商销售都要规范交税，盗版商由于缺乏进项发票，将承担更高的税负，利润空间被大幅压缩。

这些灰色产业的生产商，本身就不具备合法资质，无法为下游提供正规的进项发票。

在电商税全面实施后，销售盗版书的电商将面临困境——由于无法取得进项发票抵扣，他们必须按全额销售额缴纳高额增值税，利润空间被急剧压缩。

同时，完整的发票和税务链条，为商品**溯源机制**提供了坚实的数据基础。

一旦出现问题，执法部门可以迅速向上追踪到供货源头，使得制假售假者的隐匿成本变得极高。

这意味着，没有 [3C 认证](https://zhida.zhihu.com/search?content_id=757354493&content_type=Answer&match_order=1&q=3C+%E8%AE%A4%E8%AF%81&zhida_source=entity)的小家电、没有生产许可的化妆品、仿冒名牌的服装等都难以在正规的行业中立足。

同时，税务链条的完整也有利于[商品溯源](https://zhida.zhihu.com/search?content_id=757354493&content_type=Answer&match_order=1&q=%E5%95%86%E5%93%81%E6%BA%AF%E6%BA%90&zhida_source=entity)，让没有资质、不合规的产品难以在市场上流通。

这对消费者是保护，对正规企业更是公平。

第三，电商税有助于恢复实体和电商之间的公平竞争
-----------------------

目前实体店普遍面临的一个困境是：电商因不交税而拥有不合理的价格优势，形成 “[低价倾销](https://zhida.zhihu.com/search?content_id=757354493&content_type=Answer&match_order=1&q=%E4%BD%8E%E4%BB%B7%E5%80%BE%E9%94%80&zhida_source=entity)”。  
实体行业却必须依法纳税，成本天然偏高。

而实体经济的健康发展，其实关系到很多人的就业、收入、商铺租金和区域消费力。

如果线上线下能在公平的税负环境下竞争，两者完全可以互补共存——毕竟各自都有不可替代的优势。

最后，很多人担心电商税会导致网购涨价
------------------

确实会，但幅度有限。

增值税本质是对利润征税，而电商为了保持竞争力，很可能会选择自己消化部分税负成本。

同时，商品质量有望随之提升，形成 “优质优价” 的良性循环。

至于真正受影响的，大概是那些靠不开发票、低价冲量的主播——以后他们想靠逃税买劳斯莱斯，可能会难一点了。

一、从数据比对到实质核查的转变
---------------

**这次核查的重点对象很明确：长期 0 申报或申报收入远低于平台报送数据的电商企业**。

与前两轮仅要求说明数据差异不同，第三轮通知要求企业提供四类完整资料：销售类、采购类、资质类和全年账册。这四类资料构成了完整的业务证据链。

税务部门之所以能够精准锁定目标，得益于[金税四期系统](https://zhida.zhihu.com/search?content_id=757426641&content_type=Answer&match_order=1&q=%E9%87%91%E7%A8%8E%E5%9B%9B%E6%9C%9F%E7%B3%BB%E7%BB%9F&zhida_source=entity)与电商平台的全面对接。系统会自动标记出平台报送数据与企业申报数据差异率超过一定阈值的企业。

“差异率超过 30% 的企业，基本上都会被列入这一轮的核查名单。” 一位了解内情的税务人士表示。

二、四类资料的具体要求
-----------

**销售类资料包括但不限于**：各电商平台后台的完整销售数据、银行流水明细、支付宝 / 微信支付等第三方支付平台的交易记录。

广东一家家居用品电商被要求提供其在亚马逊、速卖通等平台的近三年完整订单报告，包括已退款订单。“我们之前只按回款净额申报，现在要解释每一笔收入的来龙去脉。” 该企业负责人表示。

**采购类资料要求更为严格**：所有采购的发票、合同、付款记录，即使是从小商品市场采购也要提供相关凭证。无法提供发票的，需要说明原因并提供替代证明。

“最头疼的是早期很多采购是从义乌小商品市场拿货，根本拿不到发票。” 一位服装电商老板坦言，“现在要一一补凭证，工作量巨大。”

**资质类资料包括**：营业执照、开户许可证、[跨境电商企业备案证明](https://zhida.zhihu.com/search?content_id=757426641&content_type=Answer&match_order=1&q=%E8%B7%A8%E5%A2%83%E7%94%B5%E5%95%86%E4%BC%81%E4%B8%9A%E5%A4%87%E6%A1%88%E8%AF%81%E6%98%8E&zhida_source=entity)、[外汇收支申报表](https://zhida.zhihu.com/search?content_id=757426641&content_type=Answer&match_order=1&q=%E5%A4%96%E6%B1%87%E6%94%B6%E6%94%AF%E7%94%B3%E6%8A%A5%E8%A1%A8&zhida_source=entity)等。

**全年账册则是重中之重**：包括总账、明细账、现金日记账、银行日记账、财务报表等所有会计资料。

三、0 申报企业的特殊风险
-------------

**长期 0 申报的电商企业在此轮核查中面临更大风险**。

税务部门的基本逻辑是：如果企业在平台有持续经营行为，却长期 0 申报，这本身就存在矛盾。一家年销售额超过 2000 万元的亚马逊卖家，连续三年 0 申报，在此轮核查中直接被认定为恶意逃税。

“企业需要证明 0 申报的合理性，比如确实处于筹建期或者有正当亏损原因。” 税务专家指出，“但如果平台数据显示持续交易而企业坚持 0 申报，很难通过核查。”

浙江一家电商企业就因长期 0 申报被要求补税并处罚款，合计金额超过 300 万元。“早知道这样，当初就应该正常申报，哪怕申报额度低一些。” 该企业负责人后悔地说。

四、如何应对第三轮税务核查
-------------

**面对第三轮核查，企业需要做好充分准备**。

首先，要立即整理所有要求的四类资料。即使资料不完整，也要尽可能提供，并准备合理的解释说明。

其次，要配合税务机关的工作。消极应对或拒绝提供资料，可能被视为拒不配合，导致更严重的后果。

最重要的是，要寻求专业帮助：电商企业的业务有其特殊性，专业财税机构能够帮助企业更好地与税务机关沟通，说明业务实质。

最后想对各位老板说
---------

**第三轮税务核查标志着电商税务监管进入新阶段**。从最初的数据比对到现在的实质核查，税务部门对电商行业的了解越来越深入，监管手段也越来越精准。
