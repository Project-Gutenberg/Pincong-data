**关于《诛仙 1》：**

其实坦白说，去年我是花钱进影院看了这部电影的，看完我豆瓣打了 3 星（及格边缘）。因为：**IP 情怀 + 港片情怀**。

金扫帚奖**《诛仙 1》**得奖词：

外表集时代光环之大成：顶流明星 + 特大 IP + 成熟工业团队，它的内核也集时代弊端之大成：无脑剧本 + 肤浅笑料 + 俗套演绎。团队的漫不经心与急功近利是显而易见，程小东用 1990 年代的旧瓶酿 2010 的新酒，酿出酸臭难耐的过期果冻。这部电影展示了中国电影的下限有多低，甚至是中国工业制造的下限有多低，票房越高，给行业的示范就越糟。

金扫帚奖**肖战**得奖词：

张小凡这个人物自始至终没有立起来，肖战无法赋予角色性格魅力，表演用力过猛，没有整体感，人物成了赶场木偶。他不知如何驾驭角色，魔化后只注重外部形态，看不见心理变化。与对手的交流缺乏节奏，互动中既没有火花也没有浪花。总体而言，肖战在影片《诛仙》中的表现不及格，最令人失望实至名归。

但，换个角度，由于我当时对影片没有过高期待，程小东的这部 1990 年代的旧瓶对我而言，反倒是一种怀旧的情怀。

而对表演，自然也没有特别的期待，**所以当时没觉得特别失望。**

2017 年暑假，情况陡然一转。2017 年暑期档，兼具杨洋刘亦菲人气号召、原著大 ip、年初剧版爆火三大优势支撑的《三生三世十里桃花》本以为可以预定暑期档票房冠军，却被横空出世的《战狼 2》打得满地找牙，最终票房仅有《战狼 2》的十分之一，打这一年起，中国电影开始了逆流量化风潮。这是电影人和观众互相选择的结果，观众用兜里的钱选择了演技过硬、制作精良、反映了群众生活、弘扬了家国情怀的好电影，让精心打磨电影质量的电影人名利双收，这也就刺激了中国电影继续走以内容为导向的正向发展道路。

2017 年以后，有四部流量电影值得我们好好谈谈，这四部电影分别见证了电影去流量化的四个里程碑。

第一部是《欧洲攻略》，实力演员做配、流量明星唱戏的模式，本来是之前流量电影稳赚不赔的路子，但这部电影遭遇到了 1.5 亿的票房惨败，也宣告了四大顶流之一的吴亦凡电影生涯的死刑，《欧洲攻略》以后，吴亦凡当歌手，上综艺，就是没有再接到电影资源。

第二部是《阿修罗》，这部电影拉胯的原因和《欧洲攻略》一样，本以为实力演员（梁家辉、刘嘉玲）+ 流量小生（吴磊）就可以稳赚不赔，但事实证明这套行不通。不过这部电影更有意思的是，制片方满脑子上个十年的制作思路，自以为用一个架空世界的虚幻设定，配上张牙舞爪的魔幻元素，就可以把电影包装成大片从而吸引到想找视觉刺激的观众入场观看，给电影上个双保险，但这一条路，早在 8 年前的《画壁》就已经证明走不通了，而且人家《画壁》还有柳岩，你《阿修罗》打算让我看谁……《阿修罗》的票房表现，我想大家也都知道，上映三天就灰溜溜地撤档，可以说也是一个笑话了。

第三部是《上海堡垒》，这完全是一部以流量时代的思路制作的电影，《上海堡垒》立项于 2013 年，开拍于 2016 年，如果这部电影放到 2016 年，鹿晗 + 江南原著的搭配，在票房上不说大爆，至少也能打平他自己当年的《盗墓笔记》，但坏就坏在这部电影是 2019 年上映的。2019 年，最有票房号召力的演员已经不是吴亦凡、鹿晗、杨洋这种流量小生了，而是吴京、沈腾、黄渤、徐峥这种真正的实力派。最让人喜欢的电影题材也不是小鲜肉谈恋爱，而是各种现实主义、家国情怀的片子。尤其是 2019 年，在《流浪地球》拉高了对中国科幻电影的期待以后，上海堡垒顶风作案，借着科幻的名头胡来，挂羊头卖狗肉，最终遭遇大败。

《上海堡垒》这部片平心而论，烂不过《逐梦演艺圈》，只是一部中规中矩的豆瓣四到五分烂片罢了，之所以受到网上从未有过的差评，我想更是因为大家那它当了一个出气筒，把这么多年被流量喂屎喂出来的怒火，一口气全还给搞流量电影这一套的人。

第四部是《诛仙 1》，《诛仙 1》同样是一部生错了时代的电影，放到 2015-2017 年，这种对大 ip 进行魔改，用流量明星担票房，吃完书粉吃演员粉的 ip + 流量电影比比皆是，比如《九层妖塔》什么的。但是大人，时代变了。2019 年这么搞不行了，你这一套可以骗一骗看完《致青春 2》看《栀子花开》的人，但你绝对骗不了看完《我不是药神》就去看《红海行动》的人。中国电影能做到多好，我们的观众已经完完全全清楚了，这个时候你的烂片就再也蒙不了人了。《诛仙 1》还有一个重要意义，那就是证明流量电影的不行。不是因为吴亦凡、鹿晗等人的流量过气，你现在换个最新鲜的男女顶流还是不行，那这就是整个流量模式的不行。

中国电影市场的流量化和去流量进程其实给我们每个人都上了一个很重要的一课，那就是你要去支持你喜欢的，反对你不喜欢的。如果资本发现喂屎就能赚钱，那他们就会一直喂屎给你。只有当你主动表示我不吃这个，我要真真正正的艺术，并用自己的钱包支持那些好好做电影的人，让这些人因为好好做电影而得到实在的收益，让其他电影人也发现好好做电影才是赚钱的唯一途径，那大家就会一起建立一个内容为导向的电影市场，当这个市场建立起来后，流量就会无从遁形，即使是资本再力捧，也无法扭转流量的颓势。

中国电影已经完成去流量化了，中国电视剧也在进入一个去流量化的时期，恐怕在电视剧的去流量化战斗中，还需要我们一起努力。

3\. 金扫帚奖是打击中国电影市场，在影视寒冬的今天更是恶毒。

对任何一个市场来说，打击淘汰市场里的劣质产品都是对市场秩序的维护。

能救电影市场的不是不能批评烂片，而是放严尺度，加大竞争，让好片驱逐烂片。相反，对烂片的纵容，才会毁掉电影市场。

就好比几十年前，毁掉北美游戏市场的是雅达利，拯救北美游戏市场的是任天堂。

前段时间，中影集团为了拯救院线，特意从自己的片库里面挑选出几部备受好评的老片，比如《中国合伙人》、《流浪地球》、《战狼 2》、《摔跤吧，爸爸》等电影送给院线作为吸引顾客的公益放映用（中影放弃自己的票房分账，将票房收入全部分给院线）。

为什么中影要选择这些备受好评的老片而不是新鲜出炉的新片作为吸引顾客重回电影院的片源呢？

因为这些老片质量已经经过了检验，对于在疫情中惊魂未定，终于下定决心进入密闭的影院观影的观众来说，可以保证他的勇气和信任值得一部好电影来回报。

如果选择新片，一旦新片质量不佳，很容易让鼓足勇气进入影院的观众们失望，这样他们大概率很久以后都不会来电影院观影。
