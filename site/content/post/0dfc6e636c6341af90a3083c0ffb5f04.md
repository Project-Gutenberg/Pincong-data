可能是对校内管理的 KPI 在走高，要稳。

加上这几年社会保守化，领导大概率看各类学生活动越来越不顺眼。

再加上咱这传统艺能，一刀切。

一方面，“丰富的校内活动” 也是 KPI，所以还得让学生去玩；

但另一方面，老登看不惯奇装异服，就算老登看得惯，拍出来的照片也不好看。

你办个学生活动想主打 “本校学子精神面貌积极向上”，结果画面里不是日本玩意就是荒废学业的游戏。你能理解，其他老登也不好理解。

再考虑一下现场管理成本。

得，把最出挑的二次元给我做咯。

咱脑回路也只能到这了，总不能领导儿子真男娘了吧？

* * *

讲真，我觉得国内二次元，挺尴尬的。

二次元是次文化里最努力鼓掌的，看个[宫崎骏](https://zhida.zhihu.com/search?content_id=751115997&content_type=Answer&match_order=1&q=%E5%AE%AB%E5%B4%8E%E9%AA%8F&zhida_source=entity)都得强调老爷子的左翼背景。连我一直觉得最魔怔的[国拟圈](https://zhida.zhihu.com/search?content_id=751115997&content_type=Answer&match_order=1&q=%E5%9B%BD%E6%8B%9F%E5%9C%88&zhida_source=entity)，规矩都能写个两页 A4 纸。

高强度出警高强度自查。

但现实里，二次元就是中老登认知里的路边一条。隔三差五漫展里来个中登现场 “抗日”，拳打脚踢之后民警也只是劝诫教育，留下被当做沙包出气的小登们一脸茫然。

你觉得自己已经内部激烈斗争划清界限，但在人家眼里，都是汉奸，顺手的事。

二次元在卖力地迎合认知里的政治正确，但其实最大的政治正确是权力层级。

就像冲进漫展里 “抗日” 的中老登，他们不是真去捍卫什么打倒什么，他们只是在享受大旗带来的权力快感：你们年轻人欢快是吧？还不是要吃我一巴掌。

* * *

高校二次元也面临类似处境。

他们觉得自己展示了学生的活力，内容也是合规的，学生也很开心，还有着相当强的现场自我管理水平，多赢。

但在校方眼里，奇装异服、潜在不正确、甚至就是 “我一老年人看不懂你们这些”，就可以毙了。

叽里咕噜说什么呢，一律停止。

* * *

再往细了说，就是：要深刻理解咱这的高校体制。

全球很多高校都会给学生一部分 “治校权”。学生在政府与校方的划定范围内，形成具有一定建议权与自主权的自治机构。

沟通是双向的，甚至能在校方资产的运营管理上有决策参与权。例如经典钉子户，[京都大学吉田寮](https://zhida.zhihu.com/search?content_id=751115997&content_type=Answer&match_order=1&q=%E4%BA%AC%E9%83%BD%E5%A4%A7%E5%AD%A6%E5%90%89%E7%94%B0%E5%AF%AE&zhida_source=entity)。

但咱这没有。

咱这从学生会到社团都纯属执行机构，不是自下而上。很多高校的社团干脆就是自上而下的校方意志，例如高校觉得自己要搞 AI，好，成立一个人工智能 XX 社。

说句暴论：你们现在看到的很多高校社团，不那么偏学习的，二次元跳舞那些，很多都是 00 年到 10 年你们师哥师姐自发成立的。

那段时间，国内高校的风气还松些。

至于现在，那就寄希望于自家校领导没那追求 “正确” 吧。

所以说，二次元真正的政治正确，就是权力层级。

对于不知情者，这或许只是一条冰冷的管理规定。但对于那些见证了过去一年风雪的人而言，这更像是一块悄然立起的墓碑，为一个曾充满生机的文化社群，画上了无声的句点。

![](https://images.weserv.nl/?url=https%3A//picx.zhimg.com/v2-a309a7d168e4e04ba355ac411d713ec9_r.jpg%3Fsource%3D1def8aca)

在禁令颁布之前，一场漫长的告别早已上演。北航校园内那个名为 “[飞梦 ACG 联盟](https://zhida.zhihu.com/search?content_id=751050582&content_type=Answer&match_order=1&q=%E9%A3%9E%E6%A2%A6ACG%E8%81%94%E7%9B%9F&zhida_source=entity)”的社团，一个曾被评为 “五星级” 的文化坐标，在过去的一年里，竟然走到了生死存亡的关头。指导老师拒绝续约，想要寻找新老师的申请，却被学院的公章牢牢卡住，求助无门。他们曾满心欢喜筹备的“[冬梦节](https://zhida.zhihu.com/search?content_id=751050582&content_type=Answer&match_order=1&q=%E5%86%AC%E6%A2%A6%E8%8A%82&zhida_source=entity)” 社庆，也在最后关头，因一句 “不可抗力”，被迫化为泡影。每一步，都走得异常沉重。

挣扎的尽头，是**未能获得今年百团招新摊位**的结局，一个曾经的庇护所，就此在制度层面失去了延续的可能。

![](https://images.weserv.nl/?url=https%3A//picx.zhimg.com/v2-a0c66b1eda84d43d43e06bf917679922_r.jpg%3Fsource%3D1def8aca)

“**[废社危机](https://zhida.zhihu.com/search?content_id=751050582&content_type=Answer&match_order=1&q=%E5%BA%9F%E7%A4%BE%E5%8D%B1%E6%9C%BA&zhida_source=entity)**”，一个常出现在虚构故事中的词汇，一旦在现实中上演，便褪去了所有戏剧性，只剩下真实的沉重感。这里没有力挽狂澜的热血桥段，也没有苦尽甘来的圆满结局。只有一份真诚而小众的热爱，在一种不容分说、无法对话的力量面前，被反复证明其渺小与脆弱时的深切无力。

如今，这股压力从一个具体的社团，蔓延到了一个完整的文化主题。校园管理规定从往年的 “禁止 Cosplay 等奇装异服”，升级为对“任何动漫相关活动” 乃至 “个人行为” 的全面禁止。这一变化标志着，被限制的已不再是特定行为，而是一整类文化在校园公开存在的正当性。

一所大学有其管理秩序和主流文化导向，尤其对于北航这样以 “[空天报国](https://zhida.zhihu.com/search?content_id=751050582&content_type=Answer&match_order=1&q=%E7%A9%BA%E5%A4%A9%E6%8A%A5%E5%9B%BD&zhida_source=entity)” 为使命的顶尖学府，强调严谨学风无可厚非。但问题在于，严谨的学术追求与包容的文化生态，是否必然是相互排斥的？

事实上，动漫及其衍生的亚文化，在当代大学生群体中，早已超越了 “幼稚” 的刻板印象，成为一种集艺术、创意、社交于一体的流行文化。在距离北航咫尺之遥的其他学府，相似的 ACG 社团却呈现出截然不同的图景：被接纳，被尊重，甚至成为校园多元文化的一张亮眼名片。这般对照，更凸显了眼前这幕悲剧的偶然性与非必然性。

它并非源于某种不可抗拒的时代潮流，**而更像是一次特定环境下的、令人扼腕的选择。**

况且，这个以青年为主的文化社群，在天性上往往并不具备强烈的对抗性。面对突如其来的压力，其成员更习惯于用戏谑、自嘲的 “梗” 来解构严肃的议题，以此消解内心的不甘；或是在同温层内，用无人听见的言语宣泄情绪。这种内向、非对抗性的特质，与自我消解的群体习惯，在不对等的权力结构中，有时反而使其被视为一个可以被轻易规训、不必过多顾虑其反弹的 “安全” 对象。

一份小小的热爱所渴求的，其实并不多。无非是一个角落的承认，一小片得以喘息的土壤，与一个不被异样目光审视的平凡空间。当这些最基本的愿望都归于沉寂，剩下的便只有无声的叹息与不解的诘问。

![](https://images.weserv.nl/?url=https%3A//pic1.zhimg.com/v2-cf666995cf572e4c7d39f9d1f7a5655a_r.jpg%3Fsource%3D1def8aca)

致意那些正在经历这场寒冬的同学们，和所有曾在角落里，试图为自己的小小世界点亮一盏灯火的灵魂。我们想要的，不是特权，只是一个平等的、被尊重的空间，让我们的正当爱好与热情，能够与这所我们深爱的大学，共同阳光地存在下去。

讲一下为什么能这么随意地被禁：

理论上的管理链条

大多数高校的社团管理体系是这样的:

```
校团委 → 学生社团联合会/学生会 → 各社团
       ↓
    指导老师(挂名)
校团委:名义上的最高管理部门
社联/学生会:具体执行管理的学生组织
指导老师:大多数情况下只是挂个名,真出事了才会出现

```

实际上的决策网络

但实际情况:

```
宣传部门:管着所有对外宣传内容的审核
学工部/学生处:管着意识形态和学生活动安全
保卫处:管着场地使用和活动审批
二级学院:有时候也要插一脚
各种上级主管部门:教育部、团中央的文件精神

```

这些部门互通有无, 一个部门觉得 "有风险", 其他部门就会跟进。

社团在学校眼里算老几?

说句难听的: 社团在学校管理者眼里, 经常是个 "可有可无" 的存在。

```
领导的优先级排序
招生就业数据 ⭐⭐⭐⭐⭐
科研经费和成果 ⭐⭐⭐⭐⭐
不出安全事故 ⭐⭐⭐⭐⭐
上级检查/评估 ⭐⭐⭐⭐
学科竞赛获奖 ⭐⭐⭐
学生社团活动 ⭐

```

社团活动如果能 "锦上添花" 当然好, 但一旦有可能 "惹麻烦", 那就是 "弃车保帅" 的第一选择。

大学社团其实处在一个管理权极不对等的位置:

```
没有正式法人资格:说白了不是正式组织
经费、场地都掌握在学校手里:要钱要地都得申请
《社团管理条例》赋予了学校极大自由裁量权
"意识形态安全"是个筐,什么都能往里装

```

所以在程序上, 只需要:

```
某领导提议 → 学工部/团委开会 → 发个通知

```

不需要征求社团意见, 不需要听证, 甚至不需要详细说明理由。

```
决策过程的草率
极有可能是:


领导:"最近二次元这块要注意,别出问题"
     ↓
下属:"那干脆全禁了吧,省事"
     ↓
团委:"收到,立即执行"

```

整个过程可能不超过半天
