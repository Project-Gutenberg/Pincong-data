所需平台：建议电脑端  
所需工具：浏览器（如Firefox、Chromium核）、文字编辑器（Windows建议[Notepad++](https://pincong.rocks/topic/Notepad%2B%2B "https://pincong.rocks/topic/Notepad%2B%2B")，Linux建议[Kate]( "https://zh.wikipedia.org/wiki/Kate")）  
  
**1：**  
进入翻车新闻官方网址 **https://www.reddit.com/user/s1refugee/posts/** （建议复制/加书签），选择最新发布的期数  
  
**2：**  
复制内容并黏贴至品葱的编辑器  
  
**3：**  
转至**<>**源码编辑，并把源码剪切至文字编辑器  
  
**4：**  
利用替换功能把  
  

```
\[url=https://preview.redd.it/
```

  
  
替换为  
  

```
\[img\]https://i.redd.it/
```

  
  
**5：**  
去掉.jpg后面的参数，如  
  

```
?width=xxxx&format=pjpg&auto=webp&s=一串字符\]\[img\]https://preview.redd.it/xxxxx.jpg?width=xxxx&format=pjpg&auto=webp&s=又一串字符\[/img\]\[/url\]
```

  
  
并以  
  

```
\[/img\]
```

  
  
代替  
  
**6：**  
把处理好的源码黏贴至品葱编辑器，在前头加上  
  

```
\[b\]搬运内容不代表品葱用户与站方立场\[/b\]  
搬运自Reddit群组r/hanren，由u/s1refugee整理。  
  
\[b\]期数：标题\[/b\]  
原地址：\[url\]原帖地址\[/url\]
```

  
  
**7：**  
在尾段加上  
  

```
上一期：（上期帖子链接）  
下一期：待发布，欢迎新搬运工加入搬运。  
  
\[i\]谢谢欣赏本次新闻！再次感谢s1refugee的汇总！\[/i\]  
\[quote\]翻车新闻品葱搬运版由@InspectorBen、@工藤新一、@Artemis、@DiskKiller3000、@维基百科、@poisson、@恶习像弹簧、@呂500、@江泽民、@MHCV01（初音未来）、@愛吃蔥油餅的瑤瑤、@（搬运员的名称） 等网友搬运。若您想订阅翻车新闻新品葱搬运版，请关注“翻车新闻”、“翻车新闻搬运”等标签获得更新。\[/quote\]  
  
\[b\]投稿地址：\[/b\]  
\[url\]https://www.reddit.com/r/hanren/\[/url\]  
  
\[b\]往期链接：\[/b\]  
>Reddit原版：  
\[url\]https://www.reddit.com/user/s1refugee/posts/\[/url\]    
  
>新品葱搬运版：  
\[url=https://pincong.rocks/topic/翻车新闻\]https://pincong.rocks/topic/翻车新闻\[/url\]  
  
>膜乎搬运版：  
\[url=https://mohu.rocks/topic/【翻车新闻】\]https://mohu.rocks/topic/【翻车新闻】\[/url\]  
  
\[i\]Be the first to share what you think!\[/i\]
```

  
  
**8：**  
如果后一期已发布，当期尾段部分段落需改为  
  

```
上一期：（上期帖子链接）  
下一期：（下期帖子链接）  
欢迎新搬运工加入搬运。
```
